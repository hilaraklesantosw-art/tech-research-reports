# 理想汽车（LI / 2015.HK）投资分析报告

**报告日期：** 2026年3月25日  
**评级：** 增持  
**12个月目标价：** 78.00港元/股（港股口径）  
**估值基准价：** 67.60港元/股（2015.HK，2026年3月24日收盘价）

---

## 一、公司概况（Company Overview）

理想汽车是中国中高端新能源乘用车市场的核心玩家之一，长期聚焦“家庭用户”场景，产品覆盖增程式SUV、纯电SUV及高端MPV。公司以“产品定义能力 + 自研智能化 + 直营销售服务 + 高压快充网络”构建差异化竞争壁垒，是港股及中概股新能源整车板块中兼具成长属性与现金储备优势的代表性标的。

截至2025年末，公司产品矩阵包括四款理想L系列增程SUV、两款理想i系列纯电SUV及一款高端MPV。渠道与补能网络方面，公司在中国拥有548家零售门店，覆盖159个城市；561家售后维修中心及授权钣喷中心，覆盖224个城市；3,907座超充站、21,651个充电桩。到2026年2月末，超充站已超过4,000座，累计交付量达到159.43万辆。

理想汽车过去几年的核心逻辑，在于其率先验证了增程路线在中国家庭SUV市场的商业化可行性，并在30万元以上家庭用车市场建立了较强品牌认知。进入2025年以来，公司战略重心由“增程稳规模”转向“纯电拓新品类 + AI智能化投入 + 全球化布局”，但该转型阶段也带来了销量波动、产品组合下沉及盈利能力承压。

我们认为，理想汽车已经从“单一路线受益者”转向“平台型新能源整车厂”的第二阶段。短期估值的核心不再是单纯交付量增速，而是公司能否在纯电产品放量、毛利率修复和现金流回正三条主线中重新建立市场信心。

## 二、财务表现（Financial Performance）

### 1. 2025年全年：收入、利润与现金流同步承压

根据公司2026年3月12日披露的2025年第四季度及全年业绩，理想汽车2025年实现总收入1,123.13亿元，同比下降22.3%；车辆销售收入1,066.83亿元，同比下降23.0%；净利润11.39亿元，同比下降85.8%；Non-GAAP净利润23.97亿元，同比下降77.5%。全年交付量40.63万辆，同比下降18.8%。

盈利能力方面，2025年车辆毛利率17.9%，较2024年的19.8%下降1.9个百分点；综合毛利率18.7%，较2024年的20.5%下降1.8个百分点；经营利润由2024年的70.19亿元转为经营亏损5.21亿元，经营利润率由4.9%转为-0.5%。现金流层面，公司2025年经营活动现金净流出86.11亿元，自由现金流为-128.17亿元，明显弱于2024年的159.33亿元和82.03亿元。

业绩大幅回落，核心原因并非单一需求疲弱，而是多重因素叠加：

- 2025年公司由增程主导向纯电拓展，产品切换阶段导致销量基数回落。
- 纯电新品导入初期，产品结构变化拉低平均售价并压缩单车利润。
- 2025年第三季度理想MEGA召回相关预计成本，对毛利率和利润端形成一次性拖累。
- 行业价格竞争加剧，终端补贴与金融优惠侵蚀盈利弹性。

### 2. 单季度趋势：Q4环比企稳，但同比仍明显承压

2025年第四季度，公司实现收入287.75亿元，同比下降35.0%，环比增长5.2%；交付10.92万辆，同比下降31.2%，环比增长17.1%；净利润0.20亿元，较2024年同期的35.33亿元大幅下滑，但较2025年第三季度亏损6.24亿元明显改善。第四季度综合毛利率17.8%，较第三季度的16.3%回升1.5个百分点，车辆毛利率回升至16.8%。

从季度变化看，公司2025年的经营轨迹可概括为“上半年弱盈利、三季度扰动、四季度止跌”。这意味着公司当前最重要的变量不再是能否守住盈利，而是2026年能否实现收入恢复增长、毛利率回到19%以上及经营现金流重新转正。

### 3. 资产负债表：现金仍是理想汽车估值底部的重要支撑

截至2025年12月31日，公司综合现金头寸为1,012亿元人民币，包括现金及现金等价物、受限现金、定期存款、短期投资及长期投资中的金融工具。即便在2025年利润和现金流显著承压的背景下，公司仍保持行业内较强的资产负债表韧性。

这意味着理想汽车虽然经历了业绩下修周期，但并未进入“融资驱动生存”的被动阶段。相反，充足现金让公司仍有能力支撑新车型研发、智能驾驶投入、补能网络建设以及最新披露的10亿美元股份回购计划。对港股投资者而言，现金实力显著降低了极端估值压缩的尾部风险。

### 4. 2026年展望：短期压力延续，中期修复取决于新品周期

公司指引显示，2026年第一季度交付量预计为8.5万至9.0万辆，同比下降8.5%至3.1%；收入预计204亿至216亿元，同比下降21.3%至16.7%。这表明公司在2026年一季度仍处于修复初段，业绩拐点并未在年初立即出现。

我们预计，2026年公司全年交付量有望恢复至47万辆左右，同比增长约15.7%；收入约1,280亿元，同比增长约14.0%；综合毛利率约19.0%；净利润约32亿元。该假设建立在以下基础上：

- 全新一代理想L9于2026年二季度上市后，有望稳定增程旗舰SUV价格带。
- i8、i6等纯电车型逐步完成渠道和用户认知爬坡，推动纯电板块从“投入期”过渡至“放量期”。
- 召回相关一次性影响消退，毛利率中枢较2025年改善。
- 股份回购及现金储备有助于稳定资本市场预期。

## 三、行业分析（Industry Analysis）

### 1. 行业总量仍增长，但增速中枢下移

根据中国汽车工业协会数据，2025年中国新能源汽车销量达到1,649万辆，同比增长28.2%；2026年行业销量预计达到1,900万辆，同比增长15.2%。这说明中国新能源汽车行业仍处于扩容期，但已经从“高增红利阶段”逐步转入“渗透率高位 + 结构竞争加剧”的阶段。

对整车企业而言，行业仍增长并不自动等于利润增长。2025年以来，新能源车市场的主要特征已从“谁能卖更多”转向“谁能在降价、补贴、智能化投入和渠道扩张中保住利润和现金流”。因此，资本市场对车企估值体系明显转向质量导向。

### 2. 理想汽车所处的是竞争最激烈的中高端家庭用车市场

理想汽车主战场集中于20万元至50万元价格带的家庭SUV与MPV市场。这一区间目前集中了比亚迪、问界、特斯拉、小鹏、蔚来、零跑、吉利银河及传统豪华品牌新能源化产品，是中国竞争最密集、产品迭代最快、用户教育最充分的细分赛道之一。

理想汽车的优势在于：

- 在家庭用户、六座/大空间SUV需求上具有较强品牌心智。
- 增程路线曾有效解决补能焦虑，积累了高质量用户基础。
- 自建销售、服务与超充网络能力较完整。
- 智能驾驶、座舱系统及整车电子电气架构具备持续自研能力。

但其劣势也非常清晰：

- 纯电转型时间晚于部分竞争者，品牌认知仍以增程为主。
- 产品带宽尚不如比亚迪/吉利等综合车企广。
- 在华为生态和小米生态持续吸引流量的环境下，理想的营销效率与技术叙事承压。
- 高端家庭SUV赛道的差异化壁垒正在被快速追赶。

### 3. 行业价格战与监管趋严并存，龙头优势更偏向综合能力

2026年初，中国监管层已针对汽车行业恶性价格竞争、低于成本销售等问题进一步加强规范。行业未来的竞争方向，预计将从单纯价格战转向“产品、补能、智能化、全球化、财务健康度”的综合竞争。

在此背景下，理想汽车虽然2025年业绩承压，但其仍具备三个相对稀缺的特征：一是曾经验证过规模化盈利；二是现金头寸充裕；三是在中高端家庭SUV领域拥有真实用户基础。因此，公司仍处于“有资格参与下一轮行业洗牌”的阵营，而非被动防守型公司。

## 四、投资推荐（Investment Recommendation）

我们给予理想汽车“增持”评级，核心逻辑如下：

- **估值已较多反映2025年业绩下行。** 以2026年3月24日港股收盘价67.60港元计算，公司市值约1,369亿港元，而截至2025年末现金头寸折合已接近估值主体的重要组成部分，市场对其2025年利润塌陷已有较充分定价。
- **2026年存在“盈利修复 + 估值修复”双重可能。** 若L9改款和纯电产品爬坡顺利，公司收入和毛利率有望同步改善，利润弹性将显著高于2025年。
- **资本配置动作释放管理层信号。** 2026年3月24日，公司董事会批准最多10亿美元股份回购计划，执行期至2027年3月31日。对经历股价大幅回调的港股标的而言，回购计划有助于改善市场对估值底部的判断。
- **现金优势保证战略主动性。** 在行业价格战尚未结束的阶段，充裕现金意味着公司可以继续投入产品研发、智能驾驶和补能网络，而不会因融资压力被迫削减战略投入。

需要强调的是，理想汽车当前并不处于基本面“高景气确认期”，而处于“盈利低谷后等待修复验证期”。因此，本报告的“增持”更偏向逆周期配置逻辑，而非高确定性成长追价逻辑。

## 五、目标价（Target Price）

我们给予理想汽车12个月目标价**78.00港元/股**。

估值方法上，我们采用2026年预期企业价值/销售额（EV/Sales）法。考虑到公司仍处于纯电车型放量初期、2026年盈利修复存在不确定性，但同时拥有较强现金储备和品牌资产，我们给予公司**2026年0.35倍EV/Sales**。基于我们对公司2026年收入约1,280亿元人民币的预测，对应目标企业价值约448亿元人民币；叠加2025年末约1,012亿元人民币现金头寸，并按约20.4亿股稀释后股本折算，对应港股合理价值约78港元/股。

相较2026年3月24日收盘价67.60港元，目标价对应约**15.4%**的上行空间。

## 六、风险因素（Risk Factors）

- **纯电车型爬坡不及预期。** 若i8、i6在订单、交付或用户口碑上低于预期，公司纯电转型节奏将被拉长。
- **增程核心车型被更强竞争者分流。** 若问界、比亚迪、特斯拉或小米等品牌在家庭SUV市场持续抢占份额，理想L系列销量和定价权可能进一步承压。
- **行业价格战超预期。** 若2026年竞争继续升级，毛利率修复路径可能明显慢于预期。
- **智能驾驶与AI投入回收期延长。** 公司持续加码VLA大模型、智能座舱和自动驾驶，若商业化兑现慢，费用率可能长期处于高位。
- **现金流恢复慢于利润恢复。** 即使会计利润改善，若库存、渠道和资本开支继续消耗现金，估值修复空间将受限。
- **政策与质量事件风险。** 产品召回、行业监管变化、补贴政策调整及海外政策壁垒，均可能影响市场情绪与经营表现。

## 七、结论

理想汽车在2025年经历了上市以来最明显的一次盈利与估值再定价，背后反映的是公司由增程优势期切换至纯电扩张期的阵痛。短期看，2026年一季度指引仍偏弱，说明基本面修复需要时间；中期看，公司在现金储备、家庭用户品牌、直营体系和智能化能力上的积累仍具备再增长基础。

站在2026年3月25日时点，我们认为市场对理想汽车的悲观预期已较充分体现在股价中，但对其2026年下半年盈利修复、资本配置改善和产品周期重启的定价仍不充分。综合行业地位、资产负债表质量与估值水平，我们给予公司“增持”评级，12个月目标价78.00港元。
