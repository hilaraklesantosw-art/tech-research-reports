# 中国工商银行股份有限公司（601398.SH / 1398.HK）深度分析报告

**报告日期：** 2026年3月24日  
**评级：** 增持  
**目标价：** 8.50元（A股口径，12个月）  
**估值基准价：** 7.25元（A股，2026年3月23日收盘价）

## 一、公司概览

中国工商银行是中国资产规模最大的商业银行，也是全球系统重要性银行中的核心中资机构之一。公司成立于1984年，2006年在上海证券交易所和香港联合交易所同步上市，当前形成了以公司金融、个人金融、资金业务、资产管理、金融市场、金融科技和全球化经营为核心的综合金融服务体系。

从经营特征看，工商银行兼具三类属性：

- 其一，是中国金融体系中的“压舱石”型资产，承担服务实体经济、支持财政金融协同和维持金融稳定的重要职能。
- 其二，是典型的高分红、低波动、大体量银行股，适合绝对收益与红利配置型资金长期持有。
- 其三，是具备一定估值修复弹性的国有大行龙头，在资本补充、息差企稳和非息收入修复阶段，股价具备防御中的进攻属性。

截至2024年末，工商银行总资产48.82万亿元，客户贷款28.37万亿元，客户存款34.84万亿元，均继续位居同业前列；对公客户1,334.86万户，个人客户7.66亿户，个人手机银行客户5.88亿户。公司绿色贷款余额突破6万亿元，制造业贷款余额4.4万亿元，普惠贷款余额2.9万亿元，均体现其在政策重点领域的领先投放能力。

我们认为，工商银行的核心投资价值不在于高成长，而在于“规模优势 + 负债优势 + 资产质量稳定 + 分红确定性 + 政策受益预期”共同构成的低波动高确定性回报框架。对于A股市场而言，该类资产在利率中枢下行、权益市场波动抬升阶段通常具备较强相对收益。

## 二、财务表现

### 1. 2024年业绩：营收承压但利润稳健，经营质量保持韧性

根据公司2024年年度报告，2024年工商银行实现营业收入8,218.03亿元，同比下降2.52%；利息净收入6,374.05亿元，同比下降2.69%；手续费及佣金净收入1,093.97亿元，同比下降8.34%；归母净利润3,658.63亿元，同比增长0.51%；净利润3,669.46亿元，同比增长0.50%。

2024年利润保持正增长的关键原因在于两点。第一，尽管LPR下行、存量按揭重定价和存款定期化压制净息差，但贷款与债券投资扩张仍对利息收入形成一定支撑。第二，信用成本明显改善，资产减值损失同比减少16.0%，对冲了收入端压力。

拆分收入结构看，工商银行2024年仍呈现明显的“大行经营特征”：

- 利息净收入仍占营业收入77.6%，盈利稳定性强，但对利率环境敏感。
- 手续费及佣金净收入占营业收入13.3%，财富管理、公募基金费改和保险“报行合一”对中收形成拖累。
- 其他非利息收益同比增长9.2%，在投资、公允价值变动和其他业务收入改善带动下，成为全年利润稳定的重要缓冲项。

### 2. 2025年前三季度跟踪：营收恢复正增长，非息收入修复明显

截至2026年3月24日，公司尚未披露2025年全年年报，最新官方财务数据为2025年第三季度报告。2025年前三季度，公司实现营业收入6,400.28亿元，同比增长2.17%；归母净利润2,699.08亿元，同比增长0.33%；利息净收入4,734.16亿元，同比下降0.70%；非利息收入1,666.12亿元，同比增长11.30%；其中手续费及佣金净收入908.68亿元，同比增长0.60%。

2025年前三季度的经营信号较为积极，主要体现在：

- 营收在2024年负增长后重新转正，反映“大行以量补价”和非息修复开始发挥作用。
- 中收同比回正，说明财富管理、托管、支付结算、投行业务等轻资本收入在低基数下逐步修复。
- 资产质量继续稳中向好，不良率较年初再降0.01个百分点，拨备覆盖率进一步提升。
- 总资产与贷款延续稳健扩张，显示其在政策性、逆周期和重点领域投放中依然是核心承接主体。

需要看到的是，净利息收益率仍处低位，意味着利润恢复仍更多依赖规模扩张、非息修复和信用成本平稳，而非净息差本身明显改善。换言之，工商银行当前并非典型高弹性银行，而是“底盘稳、增速低、分红稳”的大盘红利资产。

### 3. 分红与股东回报：高股息属性继续强化

公司2024年中期及末期现金股息合计每10股3.080元，合计派息约1,097.73亿元。以2024年归母净利润3,658.63亿元计算，分红率约30.0%。以2026年3月23日A股收盘价7.25元测算，静态股息率约4.25%。

对于国有大行而言，4%以上的股息率叠加较强盈利确定性，已具备明显的类固收配置属性。若后续公司分红率维持稳定、盈利保持小幅增长，其总回报框架将主要由“股息收益 + 估值小幅修复”构成，这也是我们给予“增持”评级的重要基础。

## 三、行业分析

### 1. 行业环境：低利率与稳信用并存，银行股投资逻辑转向红利和质量

2025年三季度末，国家金融监督管理总局披露商业银行不良贷款率为1.52%，拨备覆盖率为207.15%，资本充足率为15.36%。与之相比，工商银行同期不良贷款率1.33%、拨备覆盖率217.21%、资本充足率18.85%，均明显优于行业平均水平。

这说明在当前行业环境下，工商银行具备三项相对优势：

- 资产质量优于行业平均，风险暴露更可控。
- 拨备和资本缓冲更厚，在监管趋严和信用波动周期中更具安全边际。
- 作为国有大行龙头，负债端稳定性、低成本存款获取能力和逆周期投放能力更强。

从行业趋势看，当前中国银行业面临的主线并未改变：

- 净息差仍处下行通道，行业普遍承受资产端收益率下降压力。
- 存款成本改善具有一定滞后性，负债重定价将成为利润稳定的重要抓手。
- 非息业务修复成为估值重估的关键变量，尤其是财富管理、托管、结算、投行和资金业务。
- 银行板块的投资范式由“高增长”切换为“高分红 + 高质量 + 低估值修复”。

### 2. 国有大行比较：工商银行的优势在于规模、综合经营与系统地位

国有大型商业银行整体具备高分红和稳经营特征，但工商银行相比同业仍有更突出的龙头特征：

- 资产规模与客户覆盖居首，规模经济和系统协同优势明显。
- 对公金融、制造业金融、绿色金融、跨境人民币和托管清算等领域优势较强。
- 综合化子公司矩阵完善，工银理财、工银瑞信、工银安盛、工银投资、工银金租等牌照协同带来更完整的金融服务闭环。
- 全球网络与跨境服务能力较强，受益于人民币国际化、跨境清算与企业出海趋势。

此外，财政部已于2025年启动首批5,000亿元特别国债支持部分国有大型商业银行补充核心一级资本，表明监管层支持国有大行夯实资本、增强服务实体经济能力的政策方向明确。虽然工商银行后续资本补充的具体节奏仍需观察，但作为系统重要性最高、资产规模最大的国有大行之一，其政策受益预期仍是板块估值的重要支撑因素。

## 四、投资推荐

我们维持对工商银行的积极判断，核心逻辑如下：

- **防御价值突出。** 在低利率和宏观波动环境下，工商银行盈利波动显著低于多数周期和成长板块。
- **高股息确定性强。** 静态股息率约4.25%，在A股核心资产中仍具配置吸引力。
- **资产质量优于行业。** 不良率、拨备覆盖率和资本充足率均优于行业平均，提供估值安全边际。
- **非息收入修复开始显现。** 2025年前三季度手续费及佣金净收入转正，说明轻资本收入的下行阶段可能接近尾声。
- **政策催化存在。** 国有大行资本补充、金融支持实体经济、人民币国际化和“五篇大文章”政策推进，均有利于工行稳步扩表和估值修复。

我们预计工商银行2025-2027年将维持“收入低个位数增长、利润小个位数增长、分红稳定提升”的经营特征。该类资产不适合用高成长逻辑定价，但适合用PB、股息率与风险溢价框架定价。当前估值仍未充分反映其分红稳定性与政策受益属性。

## 五、目标价

我们采用PB估值法对工商银行A股进行估值。

核心假设如下：

- 预计2025年归母净利润约3,680亿元，同比小幅增长；
- 预计2026年归母净利润约3,780亿元，同比增长约2.7%；
- 预计2025-2026年现金分红率维持在30%左右；
- 预计2026年末每股净资产约11.3元；
- 给予2026年0.75倍目标PB。

据此，对应12个月目标价约为8.50元/股。

截至2026年3月23日A股收盘价7.25元，目标价对应约17.2%的上行空间。考虑到公司高股息回报，若将未来12个月股息收益纳入总回报测算，预期综合回报具备进一步提升空间。

之所以给予0.75倍PB，而非更高估值，原因在于行业净息差仍承压、银行板块整体估值天花板有限；但之所以不只给予0.60-0.65倍PB的传统“纯红利”估值，是因为工商银行具备更高的资产质量确定性、政策支持预期和龙头溢价。

## 六、风险因素

- **净息差继续下行。** 若贷款收益率下行快于负债成本改善，利息净收入可能继续承压。
- **信用风险暴露高于预期。** 若地产、地方融资平台、小微和零售贷款风险出清加快，可能推升减值计提压力。
- **非息收入修复不及预期。** 若财富管理、公募代销、投行业务与资本市场活跃度恢复偏弱，中收修复节奏可能放缓。
- **政策让利力度超预期。** 若金融支持实体经济进一步加码，可能对银行利润率和估值中枢形成压制。
- **资本补充节奏存在不确定性。** 特别国债注资虽为政策方向，但具体实施方案、节奏及对每家银行的影响仍待观察。
- **市场风格切换。** 若A股风险偏好显著抬升，资金可能从高股息银行股流向高弹性成长板块，导致估值修复节奏放缓。

## 七、结论

工商银行是一只典型的“高确定性、低波动、强分红”国有大行龙头标的。短期看，净息差压力仍在，但非息修复、资产质量稳定、资本实力充足以及政策支持预期，共同保障了公司基本面的稳健性。中期看，银行板块的核心定价因素将继续围绕分红、质量和政策预期展开，而工商银行在这三方面均处于同业前列。

综合考虑基本面质量、行业比较、股息回报与估值水平，我们给予工商银行“增持”评级，A股12个月目标价8.50元。
