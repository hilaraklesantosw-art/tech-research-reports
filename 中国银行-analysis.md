# 中国银行（601988.SH / 3988.HK）深度分析报告

**报告日期：** 2026年3月25日  
**评级：** 增持  
**12个月目标价：** 6.20元（A股口径）

> 注：截至2026年3月25日，中国银行2025年年度报告尚未披露。本文最新经营数据主要采用公司于2025年10月28日披露的2025年三季度业绩、2025年8月29日披露的2025年半年报，以及2025年3月26日披露的2024年年度业绩。

## 一、公司概览

中国银行成立于1912年，是中国历史最久、国际化程度最高的国有大型商业银行之一，2006年先后在香港联交所和上海证券交易所上市。公司业务覆盖商业银行、投资银行、保险、基金、租赁、资产管理等多个领域，具备较强的综合金融服务能力。

从经营特征看，中国银行与其他大型国有银行相比，最鲜明的差异化优势在于全球化网络、跨境结算能力、外汇业务壁垒及综合化经营协同。公司长期深耕跨境贸易、外汇清算、离岸人民币、国际投融资等领域，在服务“走出去”“一带一路”、外贸外资企业和人民币国际化方面具备较强先发优势。

截至2025年6月末，公司资产总额36.79万亿元，较2024年末增长4.93%；负债总额33.66万亿元，较2024年末增长4.85%。截至2025年9月末，公司不良贷款率1.24%，资本充足率18.66%，整体经营仍体现出典型国有大行“规模稳、分红强、波动低”的特征。

## 二、财务表现

### 1. 2024年业绩回顾：营收与利润稳步增长，资产质量改善

2024年，中国银行集团实现营业收入6,328亿元，同比增长1.38%；实现税后利润2,527亿元，同比增长2.58%。在行业普遍面临净息差下行压力的背景下，公司仍实现正增长，体现出较强的规模扩张能力、综合金融贡献及非息收入韧性。

| 核心指标 | 2024年 | 同比变化 |
| --- | ---: | ---: |
| 营业收入 | 6,328亿元 | +1.38% |
| 税后利润 | 2,527亿元 | +2.58% |
| 不良贷款率 | 1.25% | -0.02pct |
| 拨备覆盖率 | 200.60% | +8.94pct |
| 资本充足率 | 18.76% | 提升 |
| 全年现金股利 | 每10股2.424元（税前） | 派息率约30% |

从结构上看，2024年中国银行的业绩质量有三点值得关注：

- 第一，资产质量稳中向好，不良贷款率继续下降，说明在地产、地方融资平台等重点领域风险加速出清背景下，公司风险管控仍保持审慎。
- 第二，拨备覆盖率回升至200%以上，表明风险缓冲垫有所增强，利润质量优于单纯依靠规模驱动的增长模式。
- 第三，全年现金股利每10股2.424元，按2026年3月3日A股收盘价5.42元测算，静态股息率约4.47%，继续体现国有大行高股息配置价值。

### 2. 2025年前三季度跟踪：收入恢复增长，利润增速边际改善

根据公司2025年10月28日披露的前三季度业绩，中国银行集团实现营业收入4,921亿元，同比增长2.72%；实现本行股东应享税后利润1,777亿元，同比增长1.08%。相较2025年上半年，利润与收入变动趋势均有改善。

| 核心指标 | 2025年前三季度 | 同比变化 |
| --- | ---: | ---: |
| 营业收入 | 4,921亿元 | +2.72% |
| 本行股东应享税后利润 | 1,777亿元 | +1.08% |
| ROA | 0.70% | 基本平稳 |
| ROE | 8.98% | 小幅回落 |
| 成本收入比 | 26.90% | 维持合理区间 |
| 不良贷款率 | 1.24% | -0.01pct（较年初） |
| 拨备覆盖率 | 196.60% | 略有回落 |
| 资本充足率 | 18.66% | 保持高位 |

我们认为，2025年前三季度数据反映出公司经营的三条主线：

- 营收端已由“低位承压”切换至“温和修复”。在低利率环境下，大型银行收入普遍承压，而中国银行仍实现正增长，反映其跨境金融、金融市场及综合化业务具备一定对冲能力。
- 利润端保持低个位数增长，说明公司仍处于“息差收缩但利润可稳”的区间，盈利韧性强于部分依赖单一利差模式的同业。
- 风险指标总体平稳，不良率继续下降，显示重点领域风险暴露尚在可控范围内。

### 3. 2025年上半年补充观察：资本补充与非息修复是重要增量

2025年上半年，公司实现营业收入3,294亿元，同比增长3.61%；本行股东应享税后利润1,176亿元，同比下降0.85%，但较一季度已有改善。更重要的是，公司于2025年上半年完成1,650亿元核心一级资本补充，带动核心一级资本充足率升至12.57%，资本充足率升至18.67%，显著增强了后续信贷投放、分红稳定和抵御周期波动的能力。

同时，公司2025年上半年手续费净收入同比增长9.17%，非利息收入达到1,141.87亿元，占营业收入比重升至34.71%。这一变化说明中国银行并非完全依赖传统存贷利差，而是在财富管理、交易银行、外汇结算、金融市场等领域拥有较强创收能力。

## 三、行业分析

### 1. 宏观与行业环境：低利率延续，银行经营从“扩规模”转向“稳回报”

中国人民银行2025年前三季度金融统计数据显示，2025年9月末社会融资规模存量为437.08万亿元，同比增长8.7%。其中，政府债券余额同比增长20.2%，对社融形成显著支撑；对实体经济发放的人民币贷款余额同比增长6.4%，信贷增速中枢延续放缓。

这意味着当前银行业面临的核心环境是：

- 宏观增长保持韧性，但信贷高增长时代已结束；
- 政策重心继续偏向科技、绿色、普惠、制造业和稳地产；
- 利率中枢偏低，净息差仍是银行业共同压力；
- 银行股投资逻辑更偏向分红稳定性、资本充足性和估值修复，而非高增长叙事。

### 2. 行业监管指标对比：中国银行资产质量优于行业，资本优势更突出

国家金融监督管理总局披露，2025年二季度末商业银行整体不良贷款率为1.49%，拨备覆盖率为211.97%，核心一级资本充足率为10.93%，资本充足率为15.58%。

与之相比，中国银行2025年9月末不良贷款率为1.24%，明显优于行业平均；2025年6月末核心一级资本充足率为12.57%，显著高于行业平均；2025年9月末资本充足率18.66%，同样处于大型银行第一梯队。需要指出的是，公司拨备覆盖率约196%-200%，略低于行业平均水平，但仍显著高于监管红线，风险缓冲总体可控。

这意味着中国银行的相对优势主要体现在：

- 资产质量好于行业平均，信用成本具备稳定基础；
- 资本实力强，尤其在完成核心一级资本补充后，未来几年扩表和分红空间更从容；
- 全球化与外汇业务壁垒高，在同行同质化竞争中更具差异化。

### 3. 竞争格局判断：国有大行的估值锚在“高股息+低波动+政策属性”

中国A/H银行股长期采用PB估值框架，本质上反映的是ROE、分红率、资产质量和资本充足率的综合定价。相较股份制银行，国有大行盈利弹性通常较弱，但胜在稳定性更强、估值更低、股息率更高。

中国银行的估值逻辑较工商银行、建设银行等同业进一步多出一层“国际化溢价”：一方面，公司在跨境人民币、外汇清算、国际结算、离岸债券承销等业务上更具护城河；另一方面，人民币国际化、企业出海和外贸结构升级，为其提供了中长期业务增量。

## 四、投资逻辑与推荐

我们给予中国银行“增持”评级，核心依据如下：

- **高股息配置价值明确。** 2024年全年现金股利每10股2.424元，对应A股静态股息率约4.5%，在当前低利率环境下具备较强吸引力。
- **资本补充后安全边际提升。** 2025年上半年完成1,650亿元核心一级资本补充后，核心一级资本充足率升至12.57%，显著增强了抵御周期风险和维持分红能力的基础。
- **全球化和跨境金融具备稀缺性。** 中国银行是人民币国际化和企业出海的重要金融基础设施提供者，受益于跨境贸易、外汇与国际结算业务增长。
- **基本面韧性优于市场对国有大行的静态认知。** 2025年前三季度营收同比增长2.72%，高于市场对大型银行“零增长”的普遍预期。
- **估值仍处低位。** 以2026年3月3日A股收盘价5.42元测算，市场对公司仍主要按“高股息防御资产”定价，尚未充分反映资本补充、非息修复和跨境业务韧性。

## 五、目标价

我们采用PB估值法对中国银行A股进行估值。银行股估值的核心在于ROE、分红率、资本质量和资产质量。基于公司2025年前三季度ROE约8.98%、2025年上半年核心一级资本显著补强、盈利保持低个位数增长、分红率维持约30%的判断，我们预计公司2026年合理ROE中枢约为9.0%-9.5%，2026年末每股净资产约为8.2-8.4元。

在估值倍数上，我们给予公司2026年0.74倍目标PB。该倍数高于部分纯防御型国有大行的历史低位区间，但仍显著低于优质股份行估值水平，反映的是中国银行在跨境金融、外汇业务、资本实力和股息回报上的相对优势。

据此，对应12个月A股目标价为6.20元。相较2026年3月3日收盘价5.42元，上行空间约14.4%，若叠加股息收益，未来12个月总回报具备双位数潜力。

## 六、风险因素

- **净息差继续下行。** 若贷款重定价速度快于负债成本下行，银行息差可能继续收窄，压制收入与利润表现。
- **重点领域信用风险暴露。** 房地产、地方融资平台、部分出口链或周期行业若风险加速释放，可能抬升不良生成和拨备压力。
- **海外经营与合规风险。** 中国银行海外网络广、跨境业务多，地缘政治、制裁、反洗钱及境外监管变化均可能带来额外经营波动。
- **汇率与资本市场波动。** 外汇和金融市场业务虽提供收益弹性，但在高波动环境下也可能带来估值和非息收入扰动。
- **政策导向变化。** 若后续金融让利、资本监管、分红导向或重点领域信贷投放要求发生变化，盈利和估值中枢可能受到影响。

## 七、结论

中国银行是典型的“低估值、高股息、强资本、强全球化能力”的大型国有银行龙头。短期看，行业仍处于低利率与弱息差环境，利润高弹性难现；但中期看，公司资本补充已经落地，资产质量优于行业平均，跨境金融与外汇业务具备稀缺性，叠加国有大行估值修复与股息配置需求，具备较好的绝对收益与相对收益基础。

综合基本面、行业地位、分红确定性与估值水平，我们给予中国银行“增持”评级，A股12个月目标价6.20元。
