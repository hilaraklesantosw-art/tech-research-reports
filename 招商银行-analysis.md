# 招商银行（600036.SH / 3968.HK）深度分析报告

**报告日期：** 2026年3月24日  
**评级：** 增持  
**12个月目标价：** 52.00元（A股口径）

## 一、公司概览

招商银行成立于1987年，是中国股份制商业银行龙头之一，2002年在上交所上市、2006年在港交所上市。公司以零售金融为核心竞争优势，业务布局覆盖零售金融、公司金融、投行与金融市场、财富管理与资产管理四大板块，并拥有永隆银行、招银理财、招商基金、招银国际、招银金租等多张金融牌照，具备较强的综合金融服务能力。

从经营特征看，招商银行是典型的高ROE、高零售占比、高资本充足率的优质银行资产。其长期壁垒主要来自三方面：一是零售客户基础深厚，财富管理与AUM领先；二是低成本负债与数字化经营能力突出；三是风险偏好审慎、拨备安全垫厚，穿越信用周期的能力明显强于多数同业。

截至2024年末，公司总资产达12.15万亿元，同比增长10.19%；客户存款总额9.10万亿元，同比增长11.54%；贷款和垫款总额6.89万亿元，同比增长5.83%；零售客户总数2.10亿户，管理零售客户总资产（AUM）14.93万亿元，同比增长12.05%。公司已从单一股份行成长为兼具零售银行、财富管理平台与综合金融集团属性的头部银行。

## 二、财务表现

### 1. 2024年业绩回顾：营收承压、利润转正、基本面稳健

2024年，招商银行实现营业收入3,374.88亿元，同比下降0.48%；归属于本行股东的净利润1,483.91亿元，同比增长1.22%，全年利润增速由负转正。盈利修复的主要支撑来自规模扩张、其他非息收入改善、成本效率优化及信用成本下行。

| 核心指标 | 2024年 | 同比变化 |
| --- | ---: | ---: |
| 营业收入 | 3,374.88亿元 | -0.48% |
| 归母净利润 | 1,483.91亿元 | +1.22% |
| 净利息收入 | 2,112.77亿元 | -1.58% |
| 净手续费及佣金收入 | 720.94亿元 | -14.29% |
| 其他净收入 | 541.17亿元 | +34.13% |
| ROAA | 1.28% | -0.11pct |
| ROAE | 14.49% | -1.73pct |
| 净息差（NIM） | 1.98% | -0.17pct |
| 不良贷款率 | 0.95% | 持平 |
| 拨备覆盖率 | 411.98% | -25.72pct |
| 核心一级资本充足率（高级法） | 14.86% | +1.13pct |

从结构看，2024年公司仍处于典型的“息差下行、非息分化、信用稳定”阶段：

- 利息业务方面，净息差降至1.98%，主要受LPR下行、资产重定价和行业让利影响，但降幅已较前期趋于可控。
- 非息业务方面，手续费收入仍受基金、保险、理财代销环境拖累，但其他净收入显著增长，对全年营收形成有效对冲。
- 成本控制方面，成本收入比31.89%，同比下降1.07个百分点，反映数字化运营和精细化管理仍在释放效率。
- 资产质量方面，不良率维持0.95%的行业低位，信用成本降至0.65%，风险暴露可控。

值得注意的是，招商银行的盈利质量仍处于行业第一梯队。尽管ROE较高点有所回落，但14.49%的ROAE在A/H上市银行中仍具显著竞争力；同时，公司高级法下核心一级资本充足率14.86%，明显高于行业平均，为未来分红稳定、信贷投放和估值修复提供支撑。

### 2. 2025年前三季度跟踪：收入仍承压，盈利边际改善

根据公司2025年第三季度官方业绩摘要，2025年前三季度公司实现营业净收入2,512.82亿元，同比下降0.52%；归属于本行股东的净利润1,137.72亿元，同比增长0.52%。整体看，收入端仍受息差压制，但利润端已恢复正增长，经营韧性继续体现。

| 核心指标 | 2025年前三季度 | 同比变化 |
| --- | ---: | ---: |
| 营业净收入 | 2,512.82亿元 | -0.52% |
| 归母净利润 | 1,137.72亿元 | +0.52% |
| 净利息收入 | 1,600.42亿元 | +1.74% |
| 非利息净收入 | 912.40亿元 | -4.27% |
| 净手续费及佣金收入 | 562.02亿元 | +0.90% |
| 净息差（NIM） | 1.87% | -0.12pct |
| ROAA | 1.22% | -0.11pct |
| ROAE | 13.96% | -1.42pct |
| 零售AUM | 16.60万亿元 | +11.19% |

我们认为，2025年前三季度数据释放出三个积极信号：

- 第一，手续费收入自2022年底以来首次恢复同比正增长，说明财富管理链条正在逐步修复。
- 第二，净利息收入仍实现正增长，表明规模扩张、负债成本优化在一定程度上对冲了息差下行。
- 第三，AUM、金葵花及以上客户数、零售客群继续增长，零售护城河并未削弱。

总体而言，招商银行当前已从“高增长阶段”切换至“高质量、低波动增长阶段”。在行业普遍面临低利率和资产端回报承压背景下，公司仍有能力保持利润正增长，核心原因在于负债质量、客群质量和风控质量均优于同业。

### 3. 分红与资本回报

公司2024年度拟每股派发现金股利2.00元，对应普通股现金分红总额504.40亿元，对应2024年归母净利润的分红率约34%。在资本充足率高位、盈利稳定和监管鼓励提升投资者回报的背景下，招商银行具备维持较强现金分红能力的基础。

对于银行股估值而言，稳定分红本身就是重要定价锚。招商银行兼具较高ROE与较高分红确定性，这也是其长期享有行业估值溢价的重要原因。

## 三、行业分析

### 1. 行业环境：宽货币、低利率、重结构

2025年二季度中国人民银行货币政策执行报告显示，2025年上半年GDP同比增长5.3%，6月末社会融资规模存量同比增长8.9%，M2同比增长8.3%；5月央行下调存款准备金率0.5个百分点，并下调政策利率0.1个百分点，带动社会综合融资成本继续下行。

同时，2025年二季度末金融机构人民币贷款余额268.56万亿元，同比增长7.1%；其中普惠小微贷款同比增长12.3%，绿色贷款余额较年初增长14.4%，房地产贷款增速也较上年末有所回升。由此可见，当前银行业面临的核心特征是：

- 总量增长中枢放缓，但仍具稳定扩张空间；
- 信贷投放向科技、绿色、普惠、制造业等政策支持领域倾斜；
- 低利率环境延续，息差仍是行业最大压力来源；
- 银行股投资逻辑由“规模扩张”转向“质量、分红、估值修复”。

### 2. 行业风险格局：招商银行显著优于行业平均

国家金融监督管理总局数据显示，2025年二季度末商业银行整体不良贷款率为1.49%，拨备覆盖率为211.97%，核心一级资本充足率为10.93%。相比之下，招商银行2024年末不良率0.95%、拨备覆盖率411.98%、核心一级资本充足率14.86%，显著优于行业均值。

这意味着招商银行在以下三方面具有相对优势：

- 资产质量更稳，穿越经济波动能力更强；
- 拨备更厚，利润释放空间和抗风险能力更优；
- 资本更充足，在分红、扩表和应对监管新规方面更从容。

### 3. 行业竞争判断：优质股份行仍将获得估值溢价

在中国银行业的估值体系中，头部国有大行更多体现“高股息、低估值、稳经营”，而招商银行这类优质股份行则体现“盈利能力更强、零售转型更深、估值弹性更高”。若后续资本市场活跃度、居民财富配置需求和公募基金销售环境改善，招商银行在财富管理与轻资本业务上的弹性有望重新显现。

我们预计未来1-2年，行业难现系统性高增长，但优质银行将继续分化胜出。招商银行凭借零售、财富管理、数字化和综合金融协同优势，仍有望保持相对行业的超额收益表现。

## 四、投资逻辑与推荐

我们维持对招商银行的积极判断，核心逻辑如下：

- **优质零售护城河仍在。** 客户、AUM、财富管理体系仍在扩张，零售基因并未削弱。
- **盈利韧性强于行业。** 在息差下行周期中，公司仍保持净利润正增长，显示经营质量优势。
- **资产质量与资本水平领先。** 低不良、高拨备、高资本构成估值安全边际。
- **分红能力稳定。** 高股息确定性提升银行股配置吸引力。
- **财富管理修复具备弹性。** 若资本市场活跃度提升，公司手续费收入恢复空间更大。

基于当前基本面，我们预计公司2025-2027年归母净利润将维持低个位数稳健增长，ROE虽较历史高位回落，但仍将保持在行业领先区间。对于机构投资者而言，招商银行属于兼具防御属性与估值修复弹性的银行核心配置标的。

## 五、目标价

我们采用PB估值法对招商银行A股进行估值。考虑到：

- 2024年末公司归属于普通股股东的每股净资产为41.46元；
- 公司未来两年仍可维持约13%-14%的ROE；
- 分红率预计稳定在33%-35%区间；
- 在优质股份行中，公司应享有高于行业平均的估值溢价；

我们预计2026年每股净资产约为49.5元，给予2026年1.05倍目标PB，对应12个月目标价52.00元。

相较多数A股上市银行0.6-0.9倍PB的估值区间，该估值溢价具有合理性，反映的是招商银行更优的盈利能力、客群质量、轻资本业务优势及更强的风险抵补能力。

## 六、风险因素

- **息差收窄超预期。** 若LPR继续下行、存款定期化趋势加深，NIM可能进一步承压。
- **零售资产质量波动。** 若居民收入预期偏弱、消费金融和信用卡风险上升，零售不良生成可能抬头。
- **财富管理修复不及预期。** 若资本市场持续低迷，代销基金、保险、理财等手续费收入恢复将偏慢。
- **地产与地方债相关风险暴露。** 虽然公司风险控制相对审慎，但若重点领域信用出清加快，仍可能对拨备和利润造成扰动。
- **监管与政策变化。** 资本新规、让利实体政策、分红导向和拨备监管口径变化，均可能影响盈利与估值。

## 七、结论

招商银行已进入由“高增速”向“高质量”切换的新阶段。短期看，低利率环境仍对收入端形成压制；但中期看，公司在零售、财富管理、风控、资本和数字化方面的核心优势并未改变。与行业相比，招商银行仍是少数能够同时兼顾盈利能力、风险控制、分红回报和估值弹性的银行龙头。

综合考虑公司基本面质量、行业相对优势及估值中枢，我们给予招商银行“增持”评级，A股12个月目标价52.00元。
