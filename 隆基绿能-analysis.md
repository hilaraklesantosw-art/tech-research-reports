# 隆基绿能（601012.SH）深度分析报告

**报告日期：** 2026年3月25日  
**评级：** 审慎增持  
**12个月目标价：** 21.00元

> 注：截至2026年3月25日，公司2025年年度报告尚未正式披露。本文最新经营数据主要采用公司于2026年1月19日披露的2025年度业绩预告、2025年10月31日披露的2025年三季度报告、2025年8月23日披露的2025年半年度报告，以及2025年4月30日披露的2024年年度报告。

## 一、公司概览

隆基绿能是中国光伏产业链中最具全球竞争力的龙头企业之一，核心业务覆盖单晶硅片、电池组件、分布式及集中式光伏解决方案、场景化系统产品，以及“光伏+绿氢”延伸布局。公司长期以单晶路线和高研发投入著称，在硅片、电池组件效率提升、制造工艺优化和全球渠道建设方面具备较强先发优势。

从竞争定位看，隆基绿能并非单纯的“规模型组件厂”，而是兼具技术平台属性、全球化渠道能力与制造运营能力的综合型龙头。公司在BC电池组件、泰睿硅片、场景化产品定义以及全球品牌端具备较高辨识度。根据公司2024年年报，公司单晶硅片过去十年累计出货量全球第一，组件过去六年累计出货量稳居全球前两名。

截至2025年9月末，公司总资产1,539.72亿元，归母所有者权益573.82亿元；截至2025年6月末，公司货币资金493.03亿元，显示其在行业深度调整期仍保持较强流动性安全垫。我们认为，充足的现金与较完整的全球制造、销售网络，是公司穿越本轮光伏下行周期的重要基础。

## 二、财务表现

### 1. 2024年业绩回顾：行业价格坍塌导致公司十年来首次显著亏损

2024年，公司实现营业收入825.82亿元，同比下降36.23%；归属于上市公司股东的净亏损86.18亿元，上年同期为盈利107.51亿元；扣非后净亏损87.47亿元；经营活动现金流净额-47.25亿元；年末归母净资产608.95亿元，同比下降13.61%。

| 核心指标 | 2024年 | 同比 |
| --- | ---: | ---: |
| 营业收入 | 825.82亿元 | -36.23% |
| 归母净利润 | -86.18亿元 | -180.15% |
| 扣非归母净利润 | -87.47亿元 | -180.74% |
| 综合毛利率 | 7.44% | -10.82pct |
| 经营现金流净额 | -47.25亿元 | -158.21% |
| 研发费用 | 18.15亿元 | -20.48% |
| 研发投入合计 | 50.14亿元 | 占收入6.07% |
| 归母净资产 | 608.95亿元 | -13.61% |

拆分来看，2024年经营承压主要来自三方面：

- 行业供需错配下组件、硅片价格大幅下行，导致收入与利润率同步收缩。
- 硅片环节盈利恶化最为明显。2024年硅片及硅棒收入82.07亿元，同比下降66.53%，毛利率为-14.31%。
- 资产减值、联营企业投资收益下滑与产能利用率不足，共同放大了利润端压力。

不过，公司并非所有业务都同步恶化。2024年电站业务收入63.43亿元，同比增长67.43%，毛利率35.11%；组件及电池收入663.34亿元，仍是主收入来源，但毛利率回落至6.27%。这说明在行业底部阶段，公司电站与系统解决方案业务仍具一定对冲作用。

### 2. 2025年前三季度：亏损收窄，经营质量较2024年改善

2025年前三季度，公司实现营业收入509.15亿元，同比下降13.10%；归母净亏损34.03亿元，较上年同期净亏损64.86亿元明显收窄；扣非净亏损44.54亿元；经营活动现金流净额18.19亿元，同比由负转正；期末归母所有者权益573.82亿元，较2024年末下降5.77%。

| 核心指标 | 2025年前三季度 | 同比 |
| --- | ---: | ---: |
| 营业收入 | 509.15亿元 | -13.10% |
| 归母净利润 | -34.03亿元 | 亏损收窄 |
| 扣非归母净利润 | -44.54亿元 | 亏损收窄 |
| 经营现金流净额 | 18.19亿元 | 明显改善 |
| 基本每股收益 | -0.45元 | 改善 |
| 归母所有者权益 | 573.82亿元 | 较年末-5.77% |

我们认为，2025年前三季度的关键积极信号并不在于“已恢复盈利”，而在于：

- 亏损幅度同比收窄约48%，反映内部降本、费用压缩和减值压力边际下降开始见效。
- 经营现金流恢复为正，说明公司在库存、应付和采购节奏管理上较2024年更为审慎。
- 收入仍下降，但降幅显著窄于2024年，表明公司在行业极端价格竞争下已具备一定韧性。

### 3. 2025年上半年与全年预告：BC放量推动边际改善，但行业底部时间被拉长

2025年上半年，公司实现营业收入328.13亿元，同比下降14.83%；归母净亏损25.69亿元，同比减亏26.61亿元；经营现金流净额-4.84亿元，同比较上年同期-64.13亿元显著改善。经营层面，公司上半年实现硅片出货量52.08GW，其中对外销售24.72GW；电池组件出货量41.85GW，其中组件出货量39.57GW；BC产品上半年出货约4GW。

2026年1月19日，公司披露2025年度业绩预告，预计2025年全年归母净亏损60亿元至65亿元，扣非净亏损68亿元至74亿元。虽然全年仍亏损，但较2024年归母净亏损86.18亿元已有实质改善，说明公司处于“盈利底部之后、盈利修复之前”的过渡阶段。

值得注意的是，公司在业绩预告中明确指出，2025年四季度银浆、硅料成本上涨，叠加行业低价竞争和海外贸易壁垒，使得年度利润修复慢于市场早前预期。这意味着公司盈利恢复路径不是线性反转，而更可能呈现“收入先稳、毛利后修、利润滞后”的节奏。

## 三、行业分析

### 1. 中国光伏需求中期仍强，但制造端仍处出清期

国家能源局数据显示，2024年全国太阳能发电新增装机2.78亿千瓦，同比增长28%；截至2024年末，全国光伏累计装机达到8.86亿千瓦，同比增长45%；全年光伏发电量8,341亿千瓦时，同比增长44%。从装机端看，国内终端需求仍处高景气区间，光伏渗透率继续提升。

但制造端景气与装机端景气并未同步。工信部数据显示，2024年全国多晶硅产量超过182万吨，同比增长23.6%；硅片产量753GW，同比增长12.7%；电池产量654GW，同比增长10.6%；组件产量588GW，同比增长13.5%。同时，多晶硅、组件价格分别同比下降39.5%、29.7%，行业呈现典型“量增价减”格局。

这意味着当前光伏行业的核心矛盾并非需求不足，而是：

- 产能供给阶段性过剩；
- 同质化路线竞争激烈；
- 海外贸易壁垒提升导致全球化制造和交付复杂度上升；
- 行业盈利修复速度显著慢于装机增长速度。

### 2. 技术路线竞争从“效率追逐”走向“价值差异化”

过去两年行业主流产能快速向N型迭代，但TOPCon大规模扩张也带来了新的同质化压力。在这一背景下，BC路线的投资逻辑不再只是“效率更高”，而是是否能够形成实际溢价、稳定良率和可复制的场景化产品结构。

隆基的核心看点在于：

- HPBC2.0组件量产效率最高24.8%，定位高价值、高可靠性市场。
- 2025年上半年BC产品出货约4GW，已开始从技术展示走向商业化交付。
- 公司预计2025年末HPBC2.0电池高效产能占比超过60%，产品结构将进一步升级。

我们判断，若行业价格竞争在2026年边际缓和，具备技术差异化与品牌议价能力的企业将率先恢复盈利。隆基是这一逻辑下最值得跟踪的A股龙头之一，但前提是BC产品渗透率、良率与单位盈利改善能够持续兑现。

### 3. 港股/A股光伏龙头估值体系已从PE切换至PB与周期修复框架

在行业高盈利阶段，市场往往采用PE或PEG定价；但在当前亏损或低盈利周期，PE已失去解释力，市场更关注：

- 企业是否拥有足够的现金和资产负债表韧性；
- 技术路线是否能形成下一轮盈利修复的超额收益；
- 全球渠道、品牌与融资能力是否支持其穿越低谷。

因此，对隆基绿能这类处于盈利底部的龙头，估值更适合采用PB与中期盈利修复相结合的方法。A股投资者普遍会给予具备技术领先和龙头地位的制造龙头一定周期溢价，但这一溢价需要以现金流改善、减亏与产能结构升级为支撑。

## 四、投资逻辑与推荐

我们给予隆基绿能“审慎增持”评级，核心逻辑如下。

### 1. 龙头地位仍在，产业低谷未改公司长期竞争力

公司在品牌、全球渠道、供应链管理、海外交付与研发平台方面仍处行业第一梯队。即便2024-2025年盈利承压，龙头地位并未被根本削弱。相较中小厂商，公司更有能力承受行业价格战和海外贸易摩擦带来的冲击。

### 2. BC路线是公司未来两年最重要的估值修复抓手

在TOPCon同质化严重的背景下，BC是公司打开产品溢价和盈利差异化的关键抓手。公司已披露BC二代产品良率达成预期并顺利量产，2025年上半年BC出货约4GW，2025年末高效产能占比预计超过60%。若2026年BC产品放量顺利，并带动毛利率回升，公司估值中枢有望抬升。

### 3. 财务韧性优于多数同行，具备穿越周期能力

截至2025年6月末，公司货币资金493.03亿元；截至2025年9月末，归母净资产573.82亿元，总资产1,539.72亿元。虽然盈利持续承压，但公司尚未出现明显的流动性风险信号。2025年前三季度经营现金流转正，是一个值得重视的积极变化。

### 4. 但盈利恢复仍需时间，因此评级不宜过度激进

公司2025年度业绩预告仍为60亿至65亿元净亏损，说明光伏行业供需关系并未在2025年内完成实质修复。换言之，当前买入逻辑更接近“左侧布局龙头修复”，而非“右侧确认盈利反转”。因此，我们认可其中期价值，但维持“审慎增持”而非更高评级。

## 五、目标价

由于公司2024年与2025年均处于亏损周期，传统PE估值法参考意义有限。我们采用**2026年预期PB估值法**作为主方法，并辅以周期修复逻辑进行验证。

### 1. 估值假设

- 假设2026年行业价格战较2025年边际缓和，但仍未回到高盈利阶段。
- 假设公司2026年组件与BC产品结构继续优化，全年经营实现扭亏并回到低个位数净利率区间。
- 假设2026年末公司每股净资产约为8.0元左右。

### 2. 估值区间

考虑到公司是A股光伏龙头、BC路线具备技术溢价预期、资产负债表相对稳健，但当前仍处行业底部修复期，我们给予公司**2026年2.6倍目标PB**。

据此测算，12个月目标价为：

**目标价 = 2026年预期每股净资产8.0元 × 2.6倍PB = 20.8元**

取整后，我们给出公司**12个月目标价21.00元**。

该目标价隐含的核心判断是：市场将在未来12个月逐步从“看亏损”切换到“看修复斜率”，而非要求公司短期立刻回到历史高盈利水平。

## 六、风险因素

- **行业价格战继续超预期。** 若硅料、硅片、电池、组件价格继续低于合理成本区间，公司的减亏与扭亏节奏可能再次推迟。
- **BC放量不及预期。** 若BC产品良率、客户接受度、渠道铺设或单位盈利未能达到预期，公司的技术溢价逻辑将被削弱。
- **海外贸易壁垒加剧。** 美国、欧洲及其他主要市场的本地化政策、反倾销、关税和供应链合规要求可能提升公司的出口与全球制造成本。
- **原材料价格反弹。** 2025年四季度公司已提示银浆、硅料成本上涨。若2026年成本端继续上行而产品售价难以同步提升，盈利修复将受阻。
- **资产减值与联营投资风险。** 若行业出清时间继续拉长，公司仍可能面对设备、产能及相关投资项目的减值压力。
- **资本开支回报期拉长。** 在行业底部扩张先进产能虽有战略意义，但若需求释放慢于预期，投资回报周期可能延长。

## 七、结论

隆基绿能当前的投资价值，不在于短期业绩已经反转，而在于其作为全球光伏龙头，在行业最困难阶段仍保有技术、品牌、现金储备与全球渠道四重护城河。2024年是盈利深度探底之年，2025年体现出减亏与现金流改善，但距离高质量盈利修复仍需时间。

从A股卖方研究框架看，隆基绿能更像一只处于周期底部区域的“龙头修复标的”，适合接受中期波动、看重2026年景气与技术兑现的投资者。综合行业位置、财务韧性、BC技术进展与估值弹性，我们给予公司“审慎增持”评级，12个月目标价21.00元。
