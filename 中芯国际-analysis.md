# 中芯国际（688981.SH / 00981.HK）分析报告

**报告日期：** 2026年3月24日  
**覆盖标的：** 中芯国际集成电路制造有限公司  
**投资评级：** 增持  
**12个月目标价：** A股人民币118.00元/股  
**估值方法：** 2026E市净率法为主，结合成长性与A/H相对估值交叉验证

## 一、公司概览（Company Overview）

中芯国际是中国大陆晶圆代工龙头，也是全球领先的纯晶圆代工企业之一。公司主营业务为8英寸及12英寸晶圆代工与配套服务，覆盖逻辑、电源/模拟、高压驱动、嵌入式非易失存储、射频、CMOS图像传感器等多个工艺平台，下游应用广泛分布于智能手机、消费电子、工业与汽车、计算机与平板、连接与物联网等领域。

根据公司2025年中报披露，按2024年全球纯晶圆代工厂公开销售额口径，中芯国际位列全球第二、中国大陆第一。公司最核心的资产并不只是产能规模，而是“本土制造能力 + 多工艺平台覆盖 + 客户本土化替代趋势 + 持续资本开支形成的高进入壁垒”四项能力的组合。

截至2026年3月24日，中芯国际A股收盘价为99.25元，H股收盘价为54.85港元。按当日港币兑人民币约0.879测算，H股折人民币约48.21元，A/H溢价约105.9%，反映出A股对“国产半导体核心资产”的战略溢价显著高于港股。

我们认为，公司中期投资逻辑主要来自四点：

1. 中国半导体供应链本土化仍在推进，晶圆代工是产业链中最关键的制造环节之一。
2. 公司2025年产能利用率、出货量与收入同步修复，盈利弹性已开始兑现。
3. 2026年管理层继续给出“收入增速高于同类市场行业平均、资本开支大致持平”的指引，显示扩张节奏并未中断。
4. A/H两地上市使公司同时具备A股“国产替代核心资产”与港股“全球代工对标资产”的双重定价属性。

## 二、财务表现（Financial Performance）

### 1. 2024年：收入创历史新高，但盈利仍处修复早期

根据公司2024年年报，2024年全年收入80.30亿美元，同比增长27.0%，创历史新高；毛利率18.0%；全年产能利用率85.6%；年末月产能达到94.8万片标准逻辑8英寸等效晶圆。全年归属于公司拥有人利润为4.93亿美元。

2024年的核心特征是“收入先修复、利润后修复”。一方面，消费电子和本土客户拉动订单回暖，叠加产能爬坡，收入重新进入增长通道；另一方面，折旧摊销和新产线投入仍压制利润率，因此2024年更像是盈利弹性释放前的过渡期。

### 2. 2025年一季度：景气延续，利用率继续上行

根据公司2025年一季度业绩公告，2025年第一季度实现收入22.47亿美元，同比增长28.4%，环比增长1.8%；毛利率22.5%，较2024年第四季度基本持平；归属于公司拥有人利润1.88亿美元，同比增长161.9%。同期晶圆出货量229.22万片，同比增长27.7%，产能利用率提升至89.6%。

一季度最重要的信号不是收入本身，而是“出货量增长 + 利用率提升 + 毛利率维持高位”同时发生，说明公司新增产能并未明显稀释经营质量，反而受益于本土化订单回流与产品结构改善。

### 3. 2025年上半年：收入与利润双升，经营现金流明显改善

根据公司2025年中报，2025年上半年实现收入44.56亿美元，同比增长22.0%；税前利润5.10亿美元，同比增长95.0%；归属于公司拥有人利润3.21亿美元，同比增长35.6%；经营活动现金流净额9.09亿美元，同比增长85.8%；毛利率21.4%，同比提升7.6个百分点。

管理层在中报中明确指出，上半年盈利增长主要来自晶圆出货增加、平均售价提升及产品组合优化。同期公司仍保持世界第二大纯晶圆代工厂地位，并在上半年新增接近2万片12英寸等效月产能，说明增长并非依靠短期价格因素，而是产能和客户结构共同驱动。

### 4. 2025年前三季度与全年：盈利修复趋势确立，但四季度毛利率出现阶段性回落

公司2025年第三季度实现收入23.82亿美元，同比增长9.7%，环比增长7.8%；毛利率22.0%；归属于公司拥有人利润1.92亿美元，同比增长28.9%；产能利用率95.8%，月产能提升至102.28万片。

根据公司2026年2月10日披露的2025年第四季度业绩公告，2025年第四季度实现收入24.89亿美元，同比增长12.8%，环比增长4.5%；归属于公司拥有人利润1.73亿美元，同比增长60.7%，但毛利率回落至19.2%，主要由于折旧增加。

按公司公告口径，2025年全年未经审核收入为93.27亿美元，同比增加16.2%；归属于公司拥有人利润为6.85亿美元，同比增加39.1%；全年毛利率21.0%，同比提升3.0个百分点；全年资本开支81亿美元；年末月产能提升至105.88万片；全年出货量约970万片，年化产能利用率提升8个百分点至93.5%。

这意味着，中芯国际在2025年已经完成从“收入恢复”到“盈利恢复”的关键切换。需要注意的是，四季度毛利率环比回落并不代表需求转弱，更主要反映新产能投放后的折旧压力。对晶圆代工企业而言，这类利润率波动往往是扩产周期中的正常现象。

### 5. 资产负债表与现金流：高资本开支延续，但财务安全边际仍强

截至2025年末，公司总资产522.71亿美元，归属于公司拥有人权益214.40亿美元，总权益350.21亿美元。公司总现金在手119.36亿美元，总债务125.96亿美元，净负债仅6.60亿美元，净负债率1.9%，资本结构总体稳健。

从季度现金流看，2025年第三季度和第四季度经营活动现金流净额分别为9.41亿美元和13.44亿美元；结合上半年9.09亿美元推算，全年经营活动现金流累计约31.9亿美元。虽然与81亿美元资本开支相比，自由现金流仍为负，但这与公司当前处于“高投入换取未来产能与份额”的战略阶段相匹配，并未出现流动性风险。

## 三、行业分析（Industry Analysis）

### 1. 晶圆代工行业进入“结构性景气修复 + 区域重构”阶段

当前全球晶圆代工行业并非简单的全面景气上行，而是呈现出明显的结构性修复特征：成熟制程需求率先企稳，消费电子与边缘AI带动部分应用回暖，工业与汽车需求保持相对韧性，同时全球供应链出于地缘政治与安全考虑加快区域重构。

中芯国际在2025年中报中指出，受美国关税政策、地缘政治、终端市场修复及本土供应链替代推动，更多晶圆代工需求重新回流国内，分销商也加大备货和库存回补力度。这一判断与公司2025年全年高利用率、高出货增速相互印证。

### 2. 公司行业地位稳固，头部效应持续强化

公司披露，按2024年公开销售额口径，其位居全球第二大纯晶圆代工厂。晶圆代工本身具有极强的规模、技术、人才与资本壁垒，行业竞争并非完全价格竞争，而是围绕客户认证、工艺稳定性、平台丰富度、交付能力和持续投资能力展开。

对中芯国际而言，最大的行业红利不只是“国产替代”四个字，而是客户在风险分散和供应链安全诉求下，越来越愿意把成熟制程和部分特色工艺订单向本土龙头集中。公司一旦在高利用率条件下继续扩产，其规模效应和客户黏性会进一步增强。

### 3. 2026年行业判断：增量来自本土化与应用扩张，压力来自折旧与竞争

管理层在2026年初给出的全年展望是：在外部环境无重大变化前提下，公司2026年收入增速将高于同类市场行业平均，资本开支大致与2025年持平。该表述意味着两层含义：

1. 公司并不认为2026年会进入明显下行周期，否则不会维持高资本开支。
2. 公司也没有给出过热扩张预期，表明其对价格竞争、存储周期扰动及宏观不确定性保持谨慎。

因此，我们判断2026年行业环境更接近“总量温和增长、结构继续分化”，中芯国际作为中国大陆代工龙头，依然有望跑赢行业平均，但盈利改善幅度未必线性展开。

## 四、投资建议（Investment Recommendation）

我们给予中芯国际“增持”评级，核心依据如下：

### 1. 本土晶圆代工龙头的战略稀缺性仍在提升

中芯国际所处赛道决定了其估值不能简单用传统周期股框架理解。对中国资本市场而言，公司是少数兼具“国产替代、先进制造、科技安全、超大资本开支壁垒”四重属性的核心资产。只要本土化趋势延续，公司就具有持续获得战略溢价的基础。

### 2. 2025年经营质量改善已被验证，不再只是主题性标的

2025年公司收入增长16.2%，归母口径利润增长39.1%，毛利率提升至21.0%，年化产能利用率提升至93.5%。这意味着市场最担心的两件事已经得到阶段性缓解：

1. 新产能并未明显拖累利用率；
2. 高资本开支并未阻断盈利恢复。

公司已经从“有产能、有故事”转向“有产能、有订单、有利润弹性”的阶段。

### 3. 高资本开支虽压制短期利润率，但也构成中长期护城河

2025年资本开支高达81亿美元，短期看会带来折旧提升，并在2025年四季度体现为毛利率环比回落；但长期看，大规模资本开支正是晶圆代工行业建立份额和壁垒的必要条件。对中芯国际而言，折旧压力是扩张的副作用，也是未来更大收入体量的前置成本。

### 4. A股估值已反映战略属性，但仍有基本面上修支撑

截至2026年3月24日，公司A股对应2025年末市净率约5.4倍，显著高于H股；H股折算人民币对应市净率约2.6倍。A股估值显然已经计入较强的战略溢价，但考虑到公司2026年仍有望实现快于行业平均的收入增长，且资本市场对国产半导体核心资产的定价通常更看重稀缺性与扩张能力而非短期静态PE，因此当前A股估值并未达到不可承受水平。

### 2. 目标价测算

由于中芯国际仍处于高投入扩张阶段，静态市盈率对公司估值解释力有限，我们采用2026E市净率法为主。

截至2025年末，公司归属于公司拥有人权益为214.40亿美元。按2026E利润预测和汇率假设测算，2026年末每股净资产约为19.16元人民币。考虑到：

1. 公司是中国大陆最稀缺的晶圆代工平台型资产；
2. 2025年已经验证高利用率和利润修复；
3. 2026年收入增速仍有望高于行业平均；
4. A股对战略半导体资产普遍给予高于港股的估值溢价；

我们给予公司2026E **6.2倍PB**，对应A股12个月目标价约**118.00元/股**。

以2026年3月24日A股收盘价99.25元计算，对应约18.9%的股价上行空间。若结合中长期国产替代与资本市场风格对科技制造核心资产的偏好，目标价具备可实现性。

## 六、风险因素（Risk Factors）

### 1. 外部限制升级风险

半导体行业高度依赖设备、材料、EDA及全球供应链协同。若外部出口限制进一步升级，可能影响公司扩产节奏、设备到货、工艺迭代及客户结构。

### 2. 资本开支过高导致折旧和回报周期拉长

公司当前高资本开支模式虽有利于份额提升，但若新增产能导入速度慢于预期，折旧摊销上升可能持续压制毛利率与ROE。

### 3. 下游需求波动风险

公司收入对消费电子、智能手机、计算机及部分工业需求变化较为敏感。若终端需求恢复不及预期，或分销商补库结束，利用率与价格均可能承压。

### 4. 行业竞争加剧风险

全球晶圆代工行业头部集中度高，成熟制程和特色工艺领域竞争仍在加剧。若同业扩产、价格竞争或客户重新分配订单，可能影响公司盈利修复节奏。

### 5. 归母利润受少数股东权益波动影响

公司部分重点产线采用合资架构，少数股东权益占比较高。即使经营利润改善，归属于公司拥有人利润的释放节奏也可能与营业利润改善不同步。

## 七、结论

中芯国际已经走出单纯依赖政策预期的阶段，正在逐步证明其“高投入换高利用率、再换盈利释放”的商业闭环。2025年公司收入、出货量、利用率和归母利润同步改善，说明其在本土供应链重构中的核心地位已经开始转化为财务结果。

短期看，折旧和高资本开支仍会压制部分利润率指标；但中期看，只要国内晶圆代工需求继续向头部集中，中芯国际的规模优势、平台能力和战略稀缺性就仍将支撑估值中枢。基于2026E 6.2倍PB，我们给予公司A股12个月目标价**118.00元/股**，维持“增持”评级。
