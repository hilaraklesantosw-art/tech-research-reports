# 佛山市海天调味食品股份有限公司（603288.SH / 03288.HK）投资分析报告

**报告日期：** 2026年3月25日  
**评级：** 买入  
**12个月目标价：** 43.00元/股（A股）  
**估值基准价：** 约35.0至37.0元/股（A股，参考2026年1月至2月公开市场区间价格）

## 一、公司概况（Company Overview）

海天味业是中国调味品行业绝对龙头，核心产品覆盖酱油、蚝油、调味酱、醋及复合调味料等多个品类，兼具高品牌认知、高渠道密度和高现金流属性。公司总部位于广东佛山，A股于2014年上市，并于2025年6月19日在香港联交所主板挂牌，成为调味品行业首家“A+H”上市公司，股票代码为03288.HK。

从中国消费龙头的分析框架看，海天的核心投资价值主要来自四点。其一，酱油和蚝油等基础调味品属于高频刚需消费，需求韧性强，行业波动显著小于可选消费。其二，公司在全国范围内拥有深厚经销网络和渠道下沉能力，形成较高进入壁垒。其三，海天持续推进多品类协同和线上化布局，结构升级能力优于传统单一酱油企业。其四，公司经营现金流、ROE与分红能力长期处于A股食品饮料龙头前列。

根据公司披露，2024年海天各品类收入均实现正增长，酱油、调味酱、蚝油、其他品类收入分别为137.58亿元、26.69亿元、46.15亿元和40.86亿元，同比增速分别为8.87%、9.97%、8.56%和16.75%；线上渠道收入12.43亿元，同比增长39.78%。公司2024年产销量均超过450万吨，并于2025年1月实现高明工厂入选世界经济论坛“灯塔工厂”。我们认为，海天已经不再只是传统酱油企业，而是在向“全品类调味平台 + 极致供应链公司”演进。

## 二、财务表现（Financial Performance）

### 1. 2024年：收入与利润同步修复，盈利能力明显改善

2024年，公司实现营业收入269.01亿元，同比增长9.53%；归母净利润63.44亿元，同比增长12.75%；毛利率37.00%，同比提升2.26个百分点；净利率23.63%，同比提升0.65个百分点；加权平均净资产收益率21.76%，同比提升1.04个百分点。对于一家已经具备超260亿元收入体量的调味品龙头而言，双位数附近的利润增长和20%以上ROE，体现出极强的商业模式韧性。

从产品结构看，2024年增长并非依靠单一爆款，而是多品类共振。酱油仍是最大收入来源，占总收入比重约51%；蚝油和“其他品类”增速均较快，说明公司过去几年推动多品类扩张和SKU延展的策略已开始兑现。线上渠道接近40%的增长，也表明海天在传统经销优势之外，正补足新零售和内容电商能力。

股东回报方面，公司拟每10股派发现金股利8.60元，预计现金分红47.73亿元，对应股利支付率75.24%，创上市以来最高水平。对于A股食品饮料龙头而言，这一分红水平显著强化了其“消费蓝筹 + 高现金回报”属性。

### 2. 2025年：增长延续，利润弹性继续快于收入

2025年一季度，公司实现营业总收入83.15亿元，同比增长8.08%；归母净利润22.02亿元，同比增长14.77%；扣非归母净利润21.47亿元，同比增长15.42%；毛利率40.04%，同比提升2.73个百分点；经营活动现金流净额为-4.58亿元，上年同期为-5.54亿元。尽管一季度现金流仍受春节备货与渠道节奏影响，但利润端持续快于收入端，说明成本优化和产品结构升级仍在延续。

2025年上半年，公司实现营业收入152.30亿元，同比增长7.59%；归母净利润39.14亿元，同比增长13.35%；经营活动现金流净额15.05亿元，同比增长33.56%。以半年度口径测算，公司毛利率约40.12%，净利率约25.70%，均高于2024年全年水平。我们认为，这意味着海天在原材料成本、制造效率和费用投放效率三个维度上都在改善。

2025年前三季度，公司实现营业收入216.28亿元，同比增长6.02%；归母净利润53.22亿元，同比增长10.54%；扣非归母净利润51.55亿元，同比增长11.72%；基本每股收益0.94元，整体毛利率约39.97%。虽然三季度单季收入增速回落至2.48%，但利润仍实现3.40%的正增长，显示行业竞争加剧和餐饮需求偏弱背景下，龙头依然能够通过份额提升和成本纪律稳定盈利。

### 4. 我们的盈利预测与估值判断

结合2025年前三季度经营趋势、行业竞争格局和成本变化，我们预计：

1. 2025年全年营业收入约284亿元至287亿元，同比增长约5.5%至6.5%。  
2. 2025年全年归母净利润约69亿元至71亿元，同比增长约8.5%至12.0%。  
3. 2025年全年EPS约1.22元至1.26元。  
4. 2026年在行业需求温和复苏、多品类扩张和海外渠道推进下，EPS有望提升至1.33元至1.37元。  

我们的核心判断是，海天已进入“收入中速增长、利润稳健提速、现金回报增强”的成熟消费龙头阶段。对这类公司，估值锚不应只看收入增速，而应同时考虑其品牌壁垒、渠道控制力、资本回报率与分红能力。

## 三、行业分析（Industry Analysis）

### 1. 中国调味品行业仍是大市场、低集中度赛道

根据海天港股招股书援引的弗若斯特沙利文数据，2024年中国调味品市场规模约为4,981亿元，预计2024年至2029年复合增速约为6.4%；全球调味品市场规模约为21,438亿元，预计2024年至2029年复合增速约为5.2%。这说明调味品仍属于典型的大消费长坡赛道，既有存量刚需，也有结构升级空间。

更重要的是，行业仍然相对分散。按2024年零售额计算，海天在中国调味品市场份额约4.8%，位居第一，但龙头份额仍未达到“高度寡头化”水平，意味着未来仍有通过品类扩张、渠道下沉和行业整合继续提升份额的空间。在酱油和蚝油等核心细分市场，海天份额更高，领先优势更明显。

### 2. 行业竞争逻辑正从“铺货”转向“品牌 + 供应链 + 全渠道”

过去调味品行业的竞争核心更多是线下铺货和区域品牌扩张，但当前竞争逻辑已经升级。一方面，消费者对健康化、零添加、复合调味和便捷烹饪的需求上升；另一方面，线上即时零售、内容电商、社区零售和餐饮定制化需求，都要求企业具备更强的产品迭代和柔性交付能力。

海天在这一轮竞争中的优势较为清晰。公司具备全国化经销体系、规模化酿造能力、较强研发投入以及行业领先的数智化制造体系。与区域调味品公司相比，海天的原料采购、制造成本、品牌投放和渠道管理均有更强规模效应；与新消费小品牌相比，海天在供应链稳定性和终端触达能力上具有明显优势。

### 3. A股消费视角下，海天兼具防御、修复和国际化重估逻辑

从A股消费配置角度看，海天的属性介于高股息消费蓝筹与稳健成长消费龙头之间。其基础调味品业务具备明显防御性，盈利波动显著低于白酒、啤酒和休闲食品；同时，多品类扩张、线上化渗透和A+H双平台融资，又提供了高于传统“纯防御消费股”的成长和重估空间。

公司2025年完成H股上市后，国际化叙事有所增强。短期内海外收入占比仍不高，但资本平台国际化、品牌全球化和海外渠道拓展，为公司提供了新的估值想象空间。我们认为，海天未来更接近“亚洲调味平台型龙头”，而非单一中国酱油企业。

## 四、投资推荐（Investment Recommendation）

我们给予海天味业“买入”评级，核心逻辑如下：

- 基础盘稳健。酱油、蚝油等核心品类具备高频刚需属性，需求韧性强，收入波动小。  
- 龙头地位突出。公司在品牌、渠道、规模制造、供应链管理和经销网络密度方面均显著领先。  
- 盈利能力持续提升。2024年至2025年前三季度利润增速持续快于收入增速，毛利率中枢明显抬升。  
- 股东回报优异。2024年分红率达到75.24%，并首次建立中期分红安排，现金回报逻辑强化。  
- 国际化带来重估空间。A+H上市后，公司资本平台、品牌形象与全球渠道拓展能力得到提升。  

### 目标价测算

我们采用2026年预测EPS 1.35元，给予32倍目标PE，对应目标价43.20元；考虑到调味品板块估值切换节奏与市场风险偏好波动，取整后给出12个月目标价43.00元/股。

32倍PE并不激进，原因在于：

1. 海天属于A股稀缺的高ROE、高分红、强品牌、强现金流消费龙头。  
2. 公司盈利增速虽不再属于高成长，但确定性和现金创造能力显著优于多数可比标的。  
3. 若2026年原材料成本维持温和、行业竞争理性化，估值中枢存在进一步上修可能。

## 五、风险因素（Risk Factors）

### 1. 食品安全与品牌声誉风险

调味品属于高频民生消费品，任何食品安全、标签争议或品牌舆情事件，都可能对终端销量和估值体系带来较大冲击。

### 2. 原材料价格波动风险

黄豆、糖、包装材料等原料价格若显著上升，而终端提价传导不顺，可能压缩毛利率并影响盈利弹性。

### 3. 渠道竞争与促销加剧风险

若行业内价格竞争、渠道补贴或直播电商低价化竞争超预期，可能影响公司收入质量、经销商利润空间和品牌定位。

### 4. 需求恢复不及预期风险

若居民消费偏弱、餐饮复苏斜率不足，或家庭调味品渗透增量放缓，公司收入增速可能低于预期。

### 5. 海外扩张与A+H整合预期不及预期

公司已完成H股上市，但海外业务从资本平台国际化到收入利润显著贡献仍需时间。若国际化推进节奏偏慢，市场对新增估值逻辑的期待可能回落。

---

## 六、结论（Conclusion）

海天味业当前处于典型的“成熟龙头再提质”阶段。公司已经跨过单纯依靠渠道扩张做大收入的阶段，转而依靠品牌力、全渠道运营、极致供应链和多品类协同实现更高质量增长。2024年及2025年前三季度数据表明，公司盈利能力、现金回报和经营质量均在稳步提升。

从投资框架看，海天的吸引力不在于短期高弹性，而在于其兼具防御属性、盈利确定性和中长期份额提升空间。对于希望配置中国消费核心资产的投资者而言，海天味业仍是值得中长期关注的优质标的。我们给予“买入”评级，12个月目标价43.00元/股。
