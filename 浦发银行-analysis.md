# 浦发银行（600000.SH）深度分析报告

**报告日期：** 2026年3月26日  
**评级：** 增持  
**12个月目标价：** 13.50元（A股口径）

> 注：截至2026年3月26日，浦发银行2025年年度报告尚未披露。本文最新经营数据主要采用公司于2025年10月30日披露的2025年三季度报告、2025年8月27日披露的2025年半年度报告摘要，以及2025年3月28日披露的2024年年度报告摘要。

## 一、公司概览

浦发银行成立于1992年，1999年在上海证券交易所上市，是中国12家全国性股份制商业银行之一，也是上海国资体系内最重要的金融平台之一。公司总部位于上海，业务覆盖公司金融、零售金融、金融市场、投资银行、资产管理、托管、离岸与跨境金融等多个领域，具备较完整的综合化经营牌照布局。

与头部股份行相比，浦发银行的投资逻辑并非“高成长零售银行”，而是“经营修复中的低估值股份行”。其核心看点在于三方面：一是立足上海和长三角核心区域，在对公业务、交易银行、跨境结算、科创金融和供应链金融方面具备较强基础；二是近两年围绕“数智化”战略和“五大赛道”持续重塑业务结构，信贷投放效率和负债成本管理明显改善；三是资产质量经历多年出清后进入改善通道，不良率和拨备覆盖率同步优化，利润释放弹性开始显现。

截至2025年9月末，公司资产总额为9.89万亿元，较2024年末增长4.55%；贷款总额约5.68万亿元；归属于母公司普通股股东的每股净资产为22.40元。不良贷款率降至1.29%，拨备覆盖率升至198.04%，整体经营已从“风险消化期”逐步转向“修复兑现期”。

## 二、财务表现

### 1. 2024年业绩回顾：收入承压但利润显著修复

2024年，浦发银行实现营业收入1,707.48亿元，同比下降1.55%；实现归属于母公司股东的净利润452.57亿元，同比增长23.31%；扣非归母净利润442.07亿元，同比增长30.47%。在净息差下行和行业整体营收偏弱的背景下，公司利润大幅回升，显示修复并非来自单一会计扰动，而是由规模扩张、负债成本下行、投资收益改善、信用成本下降共同驱动。

| 核心指标 | 2024年 | 同比变化 |
| --- | ---: | ---: |
| 营业收入 | 1,707.48亿元 | -1.55% |
| 归母净利润 | 452.57亿元 | +23.31% |
| 扣非归母净利润 | 442.07亿元 | +30.47% |
| 基本每股收益 | 1.36元 | +27.10% |
| ROA | 0.50% | +0.08pct |
| ROE | 6.28% | +1.07pct |
| 净利差 | 1.37% | -0.12pct |
| 净利息收益率（NIM） | 1.42% | -0.10pct |
| 成本收入比 | 29.16% | -0.49pct |
| 不良贷款率 | 1.36% | -0.12pct |
| 拨备覆盖率 | 186.96% | +13.45pct |
| 每10股现金分红 | 4.10元（含税） | 分红率30.16% |

从结构上看，2024年公司经营质量改善主要体现为：

- 贷款增长较快。年末贷款总额5.39万亿元，同比增长7.45%，高于存款同比增速3.24%，体现公司在重点赛道和重点区域信贷投放更积极。
- 负债成本改善。年报披露集团存款付息率同比下降19bp，对冲了资产端重定价压力，是净利息收入企稳的重要基础。
- 非息收入形成支撑。剔除上年出售上投摩根股权的一次性因素后，营业收入实际同比增长0.92%，说明非息业务与投资交易收入改善对营收形成有效缓冲。
- 风险成本回落。公司加大不良处置力度，2024年实现不良资产现金清收378亿元，信用减值损失及其他资产减值损失同比下降9.61%，直接释放利润弹性。

整体而言，2024年是浦发银行基本面拐点进一步确认的一年。尽管ROE仍明显低于招商银行等优质股份行，但从利润增速、资产质量、拨备和成本效率看，公司已完成从“被动修复”向“主动优化”的过渡。

### 2. 2025年前三季度跟踪：修复趋势延续，盈利增速进一步抬升

2025年前三季度，公司实现营业收入1,322.80亿元，同比增长1.88%；实现归母净利润388.19亿元，同比增长10.21%。其中，2025年上半年营收和净利润分别同比增长2.62%和10.19%，三季度单季营收同比增长0.31%，归母净利润同比增长10.29%，显示修复趋势仍具持续性。

| 扣非归母净利润 | 389.00亿元 | +14.54% |
| 基本每股收益 | 1.28元 | +13.27% |

| 成本收入比 | 27.53% | -0.47pct |
| 资产总额 | 9.89万亿元 | 较年初+4.55% |

| 拨备覆盖率 | 198.04% | 较年初+11.08pct |

前三季度的积极信号主要包括：

- 营收重新转正，说明公司在低息差环境下已具备一定收入韧性。
- 利息净收入占营业收入比重提升至67.74%，表明核心盈利仍由传统存贷业务支撑，盈利稳定性好于依赖交易波动的模式。
- 存款总额较年初增长9.19%，明显快于资产扩张，说明“存款立行”和负债精细化管理取得成效。
- 不良贷款余额较年初继续下降，拨备覆盖率接近200%，表明风险改善不仅体现在比例口径，也体现在绝对额控制。

需要看到的是，浦发银行的盈利恢复仍建立在“低ROE修复”基础上，当前ROE水平仍处行业中游偏下区间，估值修复空间存在，但向高估值银行切换仍需更长时间和更持续的盈利兑现。

## 三、行业分析

### 1. 行业环境：银行板块仍处低利率、弱息差、高分红阶段

2025年以来，中国银行业整体仍运行在低利率环境中。LPR下行、存量按揭和对公贷款重定价、实体融资成本持续下降，对银行息差形成系统性压力。与此同时，监管层持续推动支持实体经济、化解重点领域风险和稳定房地产、地方债务链条，银行业整体呈现“规模稳定扩张、盈利低速增长、资产质量边际改善、分红属性强化”的典型特征。

对于A股银行板块而言，当前最重要的定价因子主要有四个：

- 净息差能否企稳，决定收入端是否能止跌。
- 不良生成与拨备消耗节奏，决定利润修复的可持续性。
- 资本充足与分红能力，决定高股息配置价值。
- 区域经济禀赋与业务结构，决定同业分化程度。

股份制银行相较国有大行普遍具备更强的经营弹性，但同时也面临更高的信用成本波动和更明显的息差压力。在这一背景下，市场更偏好两类银行：一类是招商银行这类高ROE、高质量龙头；另一类则是浦发银行这类处于改善周期、估值显著偏低、利润弹性开始释放的修复型标的。

### 2. 浦发银行的行业定位：上海区位优势明显，修复逻辑强于成长逻辑

浦发银行的相对竞争优势主要体现在：

- **区域禀赋优质。** 公司深耕上海、长三角和沿海发达地区，在科创、先进制造、贸易结算、供应链和跨境金融等领域具备先发基础。
- **对公与交易银行能力较强。** 与零售驱动型银行不同，浦发银行在公司金融、资金业务、同业和财资金融方面积累较深，更受益于产业升级、基建链和科创企业融资需求。
- **国资背景提升经营稳定性。** 大股东结构稳定，上海国资属性较强，在重大区域战略、金融资源整合和信用背书方面具备一定优势。

但短板同样清晰：

- 零售业务护城河不及头部股份行，财富管理和轻资本中收能力仍偏弱。
- 历史包袱导致市场对其资产质量和盈利稳定性的风险偏好仍偏保守。
- 目前ROE仅处6%左右水平，距离优质银行10%以上的盈利能力尚有明显差距。

因此，浦发银行的投资框架更适合用“估值修复+股息回报+基本面改善”去理解，而不应简单套用高成长银行逻辑。

## 四、投资建议

我们给予浦发银行“增持”评级，核心逻辑如下：

- **基本面修复已经兑现，不再停留于预期层面。** 2024年归母净利润同比增长23.31%，2025年前三季度继续增长10.21%，说明经营改善具备连续性。
- **资产质量改善趋势明确。** 2024年末不良率较上年下降12bp，2025年9月末进一步降至1.29%；拨备覆盖率由173.51%升至198.04%，风险缓冲垫持续增厚。
- **负债成本改善有助于缓释息差压力。** 在行业普遍承压背景下，公司通过存款结构优化和成本管控提升了经营韧性。
- **估值安全边际较强。** 以公司2025年9月末每股净资产22.40元测算，即使给予明显低于优质股份行的估值水平，仍可支撑高于当前账面折价逻辑的合理目标价。
- **分红回报具备配置吸引力。** 2024年度拟每10股派现4.1元，对长期资金和红利策略投资者具备一定吸引力。

从配置属性看，浦发银行更适合两类资金：一是寻求低估值金融股修复机会的价值型资金；二是兼顾股息与估值弹性的中长期绝对收益资金。若后续2025年年报继续验证利润双位数增长、息差降幅收窄、资产质量持续改善，公司估值中枢仍有上修空间。

## 五、目标价

我们采用PB估值法对浦发银行A股进行估值。考虑到：

- 公司2025年9月末归属于母公司普通股股东的每股净资产为22.40元；
- 2024年ROE为6.28%，2025年前三季度加权平均ROE为5.67%，全年年化后有望处于6.5%左右；
- 公司目前处于修复阶段，估值应高于风险出清期低位，但仍应显著低于招商银行等高ROE股份行；
- A股股份制银行在弱息差周期中，合理估值中枢通常围绕0.45-0.75倍PB波动，修复型银行适合采用中枢偏上的保守估值。

基于此，我们预计公司2026年末每股净资产约为23.3元，给予2026年0.58倍目标PB，对应12个月目标价13.50元。

这一目标价反映的是“低估值修复”而非“高成长溢价”：其上行基础来自资产质量持续改善、利润修复延续和分红回报支撑，而非ROE快速跃升。若后续零售转型、中收恢复和资本效率提升超预期，估值仍存在进一步上修可能。

## 六、风险因素

- **净息差继续下行。** 若LPR持续下调、贷款重定价快于负债成本下行，公司利息净收入改善可能弱于预期。
- **重点领域信用风险反复。** 房地产、地方融资平台、部分中小民企和周期行业若风险再度暴露，可能拖累不良生成和拨备消耗。
- **非息收入波动。** 金融市场收益、手续费及佣金收入恢复若低于预期，可能影响营收修复节奏。
- **零售转型不及预期。** 若零售客户经营、财富管理和消费金融业务推进较慢，公司估值重估空间会受到限制。
- **监管与政策变化。** 分红导向、资本监管、拨备政策及支持实体让利要求若发生变化，可能影响盈利和估值中枢。

## 七、结论

浦发银行并非当前A股银行板块中经营质量最优的龙头，但它是修复趋势较清晰、估值安全边际较高、利润弹性正在兑现的典型股份行代表。短期看，行业低利率环境仍会限制其估值快速抬升；中期看，若公司继续兑现“收入企稳、利润增长、资产质量改善、分红稳定”四条主线，其估值折价有望继续收敛。

综合基本面趋势、行业位置、分红能力与估值水平，我们给予浦发银行“增持”评级，A股12个月目标价13.50元。
