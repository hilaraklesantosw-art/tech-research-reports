# 中国平安保险（集团）股份有限公司（601318.SH / 02318.HK）投资分析报告

**报告日期：** 2026年3月24日  
**评级：** 买入  
**12个月目标价：** 76.00元/股（A股口径）  
**估值基准价：** 约62.00元/股（A股，参考2026年3月上旬公开市场价格）

---

## 一、公司概况（Company Overview）

中国平安是中国少数同时具备寿险、财险、银行、资管、医疗养老与金融科技平台能力的综合金融集团，也是A/H两地市场最具代表性的保险龙头之一。公司以寿险及健康险为核心利润来源，以财险和银行构成稳定现金流与客户入口，并通过“综合金融+医疗养老”战略强化客户经营深度。

从A股和港股投资框架看，中国平安兼具三重属性：其一，是中国寿险行业价值修复的核心受益标的；其二，是高股息、强现金回报的金融权重股；其三，是具备医疗养老生态延展能力的稀缺综合金融平台。相较纯寿险公司，平安的估值锚不仅在新业务价值（NBV）增长，也在于客户经营效率、投资端弹性、分红持续性及医疗养老生态变现能力。

截至2024年末，公司实现个人客户数2.42亿，较年初增长4.7%；持有集团内4个及以上合同的客户占比25.6%，留存率达98.0%。截至2025年9月末，零售客户已接近2.50亿，合同数/客提升至2.94个。我们认为，这意味着平安已经从传统保险销售型机构，进一步演进为以客户终身价值管理为核心的综合金融服务商。

## 二、财务表现（Financial Performance）

### 1. 2024年：利润显著修复，寿险价值增长重新成为主线

根据公司2024年全年业绩公告，2024年中国平安实现归母营运利润1,218.62亿元，同比增长9.1%；归母净利润1,266.07亿元，同比增长47.8%；营业收入11,413.46亿元，同比增长10.6%；全年现金股息2.55元/股，同比增长5.0%，按归母营运利润口径计算的现金分红率为37.9%。

2024年利润高增的核心驱动来自三方面：一是寿险改革持续兑现，新业务价值恢复较快增长；二是权益市场与投资收益改善，带动投资端表现修复；三是财险综合成本率优化、盈利能力提升。2024年寿险及健康险新业务价值285.34亿元，可比口径下同比增长28.8%；平安产险综合成本率98.3%，同比优化2.3个百分点；保险资金综合投资收益率5.8%，同比提升2.2个百分点。

### 2. 2025年前三季度：经营质量继续改善，成长斜率优于同业

截至2025年9月末，公司实现归母营运利润1,162.64亿元，同比增长7.2%；归母净利润1,328.56亿元，同比增长11.5%；营业收入9,016.68亿元，同比增长4.6%；归母股东权益9,864.06亿元，较年初增长6.2%。

寿险板块仍是最重要的增长引擎。2025年前三季度，寿险及健康险新业务价值357.24亿元，同比增长46.2%；代理人渠道NBV同比增长23.3%，人均NBV同比增长29.9%；银保渠道NBV同比大增170.9%，显示“报行合一”背景下行业费用纪律改善、头部险企渠道效率和产品竞争力同步上行。财险业务方面，原保险保费收入2,562.47亿元，同比增长7.1%，综合成本率97.0%，同比优化0.8个百分点。投资端方面，保险资金投资组合非年化综合投资收益率5.4%，同比提升1.0个百分点。

银行业务对集团估值既是拖累也是稳定器。2025年前三季度，平安银行实现净利润383.39亿元，不良贷款率1.05%，较年初下降0.01个百分点，拨备覆盖率229.60%。在净息差承压、行业普遍去风险和降成本的环境下，平安银行资产质量保持平稳，有助于抑制集团估值折价进一步扩大。

### 4. 我们的盈利与估值判断

基于2025年前三季度业绩趋势，我们预计公司2025年全年归母营运利润约1,300亿至1,330亿元，归母净利润约1,420亿至1,470亿元；预计2026年营运利润仍有望实现中高个位数增长。核心依据在于：

1. 寿险新业务价值仍处于修复通道，且渠道结构改善明显。  
2. 财险承保盈利保持稳定，非车险和风险减量服务有望继续贡献利润弹性。  
3. 投资端在利率低位环境下仍面临波动，但高股息权益、长久期债券与另类资产配置有助于平滑收益。  
4. 银行业务虽难成为高增长来源，但资产质量总体可控，对集团利润形成稳定支撑。  

## 三、行业分析（Industry Analysis）

### 1. 中国保险业仍处于“负债端改善、资产端分化”的阶段

根据监管系统公开披露的行业数据，2025年中国保险业原保险保费收入约6.12万亿元，同比增长7.43%，其中人身险约4.65万亿元、财产险约1.47万亿元。行业增长仍然稳健，但驱动力正在变化: 传统储蓄型产品高增阶段已过，价值率改善、渠道效率优化和保障型需求回升成为新阶段核心变量。

寿险行业层面，低利率环境一方面压缩投资收益中枢，另一方面提升居民对长期保障、年金和健康险产品的配置需求。监管推动产品定价机制优化、持续推进“报行合一”，本质上有利于头部公司通过产品、服务、品牌和渠道能力获取份额，并改善费用率。中国平安在品牌、代理人质量、银保合作深度和医疗养老服务协同方面均处于行业前列，因此更有条件把行业“规模竞争”转化为“价值竞争”。

财险行业方面，车险是基本盘，非车险是中期弹性。新能源车险、农业保险、责任险、健康险及企业风险管理服务，正成为财险结构升级的重要方向。中国平安在车险市场具备强品牌和定价、风控、理赔科技能力，且综合成本率持续优于市场平均水平，因此其财险业务盈利确定性显著强于一般综合险企。

### 2. 中国平安的行业地位在于“综合能力”而非单一份额

平安的核心竞争力不只是寿险保费规模或财险市占率，而是其“多业务协同 + 客户交叉销售 + 医疗养老服务赋能”形成的复合护城河。2025年上半年，享有医疗养老生态权益的客户贡献了接近70%的寿险新业务价值；截至2025年9月末，享有相关服务权益的客户人均合同数约3.38个、人均AUM约6.34万元，分别为非权益客户的1.6倍和4.0倍。这意味着平安的医疗养老布局已不再是概念投入，而是开始实质性提升客户价值和销售效率。

横向比较看，若以中国A/H保险板块估值体系衡量，中国平安相较纯寿险标的拥有更稳定分红能力，相较纯财险标的拥有更强寿险弹性，相较银行股拥有更高成长性。我们认为这正是其应享受估值溢价的根本原因。

## 四、投资推荐（Investment Recommendation）

我们给予中国平安“买入”评级，核心逻辑如下：

- 寿险价值修复趋势明确。2024年与2025年前三季度NBV分别增长28.8%和46.2%，反映代理人改革、银保效率提升和产品结构优化持续兑现。
- 财险盈利能力稳健。综合成本率连续改善，承保利润弹性在行业中处于领先水平。
- 投资端具备向上弹性。公司保险资金体量大、配置能力成熟，权益市场改善阶段对利润和净资产弹性较大。
- 分红吸引力强。2024年全年股息2.55元/股，对应当前A股价格的股息率约4%左右，具备较强防御属性。
- 估值仍具修复空间。以2025年9月末归母股东权益9,864.06亿元测算，对应当前A股市净率仅约1.15倍至1.20倍，低于公司在寿险高景气修复周期中应有的估值中枢。

## 五、目标价（Target Price）

我们采用分部视角下的P/B估值法对A股进行定价。

一方面，中国平安的估值核心仍由寿险内含价值创造能力、净资产增长能力和股息回报能力决定；另一方面，银行业务、财险业务和医疗养老生态使其估值不宜简单对标单一寿险公司。基于2025年全年及2026年经营趋势，我们给予公司2026年目标P/B约1.35倍至1.40倍。按照我们对2025年末每股净资产约54元至55元的判断，对应12个月合理价值区间约73元至77元/股。

综合考虑公司在行业内的龙头地位、NBV高增长持续性、分红稳定性与当前估值安全边际，我们给出A股12个月目标价 **76.00元/股**。相较约62.00元/股的估值基准价，隐含上涨空间约22.6%。

## 六、风险因素（Risk Factors）

- **资本市场波动风险。** 权益市场若显著回调，将影响投资收益、净利润表现和净资产增速。
- **长端利率下行风险。** 若长期利率继续下探，寿险负债成本与资产再配置压力可能上升，影响估值中枢。
- **寿险改革不及预期。** 若代理人产能提升放缓、银保渠道价值率回落，NBV高增长可能难以持续。
- **银行资产质量波动风险。** 若宏观信用环境走弱、地产或地方平台相关风险暴露加大，平安银行利润与资本消耗或承压。
- **医疗养老生态变现进度慢于预期。** 若“保险+服务”模式在客户转化、ARPU提升或盈利模式上兑现较慢，市场对战略溢价的认可度可能下降。
- **监管政策变化风险。** 包括产品预定利率、渠道费用监管、资本规则和保险资金运用政策变化等，均可能对盈利和估值产生影响。

## 七、结论

中国平安当前所处阶段，已不是单纯的保险股估值修复逻辑，而是“寿险负债端改善 + 财险稳健盈利 + 投资端弹性 + 高股息 + 医疗养老战略协同”的多因子共振。2024年利润修复奠定基础，2025年前三季度则进一步验证增长质量优于行业平均。

站在2026年3月时点，我们认为市场对平安“高分红防御属性”已有一定认知，但对其寿险价值修复持续性、综合金融协同效率和医疗养老生态带来的长期估值抬升，仍未充分定价。因此，我们维持积极判断，给予“买入”评级，A股12个月目标价76.00元/股。
