# 紫金矿业（601899.SH / 02899.HK）深度分析报告

## 一、核心观点

**评级：买入**  
**A股目标价：人民币42.00元**  
**对应2026E市盈率：约18.3倍**  
**参考股价：人民币36.75元（A股，2026年3月12日收盘）**  
**预期空间：约14.3%**

我们维持对紫金矿业的积极判断。公司已从传统资源股成长为兼具全球化资源配置、内生扩产能力和资本运作能力的综合性金属矿业龙头，核心逻辑在于：  
1. 金、铜双主业同步处于高景气区间，价格与产量形成共振；  
2. 2025-2026年进入新一轮产量释放期，矿产金、矿产铜、锂当量碳酸锂增长弹性较强；  
3. 公司在全球矿业并购、建设、运营和成本控制方面已形成显著体系化优势；  
4. A/H双平台融资与资产证券化能力强化，有助于支撑中长期资源扩张和估值重估。

## 二、公司概览

紫金矿业是中国领先、全球排名前列的综合性金属矿业公司，主营覆盖铜、金、锌（铅）、银、锂、钼等矿产资源的勘探、开发、冶炼及相关工程技术服务。截至2024年末，公司总股本为265.78亿股，其中A股205.89亿股、H股59.89亿股。公司在全球19个国家、5大洲拥有30多个大型矿业项目，资源布局兼具中国、西非、中亚、南美及塞尔维亚等重点矿业区域。

从行业地位看，公司已跻身全球头部金属矿企。根据公司披露，按2025年产量口径，矿产铜、矿产金、矿产锌全球排名分别预计位居前四至前五；在中国市场，公司矿产铜、矿产金、矿产锌、矿产银产量均处于领先地位。紫金矿业已不再是单一金价或铜价映射标的，而是具备多金属组合、全球资源资产和持续扩产能力的“成长型资源平台”。

## 三、财务表现

### 1. 2024年业绩：盈利创历史新高

2024年，公司实现：

| 指标 | 2024年 | 同比 |
| --- | ---: | ---: |
| 营业收入 | 3,036.40亿元 | +3% |
| 税前利润 | 480.78亿元 | +54% |
| 归母净利润 | 320.51亿元 | +52% |
| EBITDA | 632.00亿元 | +36% |
| 经营活动现金流净额 | 488.60亿元 | +33% |
| 基本每股收益 | 1.21元 | +51.25% |

2024年公司矿产品量价齐升，矿产铜产量107万吨、矿产金73吨、矿产锌（铅）45万吨、矿产银436吨。其中矿产铜和矿产金分别同比增长6%和8%。在全球矿业成本普遍上行背景下，公司铜精矿与金精矿销售成本分别下降4.3%和0.43%，体现出较强的成本控制能力。

利润质量同样较高。2024年经营现金流净额达488.60亿元，显著高于归母净利润；期末总资产3,966亿元，归母净资产1,398亿元，资产负债率降至55%，表明公司在高强度扩张周期内仍保持较强现金创造能力和财务韧性。

### 2. 股东回报提升

公司2024年度现金分红总额首次超过100亿元。按已披露方案，全年合计分红约100.99亿元，现金分红比例约31.5%。对A股资源股而言，这一分红强度已具备较强吸引力，也有助于改善传统周期股“高波动、低回报”的估值折价。

### 3. 2025年经营趋势：高景气延续

2025年上半年，公司实现营业收入1,677亿元，同比增长11.5%；归母净利润233亿元，同比增长54.4%；ROE为16.11%，同比提升3.13个百分点；基本每股收益0.877元，同比增长52.79%。上半年矿产铜总产量56.69万吨、矿产金132.4万盎司，显示公司在核心品种上延续强增长。

更关键的是，公司在2026年2月披露的三年产量规划中给出2025年初步经营数据：  
- 2025年营业收入约3,450亿元；  
- 税前利润约800亿元；  
- 归母净利润约510亿至520亿元；  
- 经营活动现金流约730亿元；  
- 总资产约5,100亿元。

这意味着相较2024年，公司2025年归母净利润预计再增长约59%至62%，盈利中枢实现显著抬升。我们认为，2025年利润大幅增长主要受益于三点：  
1. 金价中枢显著上移；  
2. 矿产金产量由73吨提升至约90吨；  
3. 铜、锂等增量项目并表及放量贡献增强。

## 四、行业分析

### 1. 黄金板块：避险与央行需求支撑估值中枢

黄金板块的核心驱动仍是全球地缘政治不确定性、美元实际利率下行预期及全球央行持续增配黄金。对A/H两地市场而言，黄金股通常兼具“资源属性”和“宏观对冲属性”，在风险偏好波动阶段相对更易获得资金关注。

紫金矿业相较纯黄金股的优势在于，公司并非单纯依赖金价上涨带来的估值重估，而是具备矿产金产量快速增长的内生逻辑。公司规划2025年矿产金90吨，2026年提升至105吨，2028年进一步达到130-140吨。对比2024年73吨，公司未来三年矿产金增长斜率较高，意味着公司对金价上涨的盈利放大效应显著强于多数成熟型黄金企业。

### 2. 铜板块：能源转型与供给约束形成中期支撑

铜仍是全球能源转型、电网投资、电动车及AI基础设施扩张的重要基础金属。供给端受制于大型项目投放节奏、资源国政策扰动及矿石品位下降，行业中长期紧平衡格局未变。

紫金矿业是少数兼具资源储量、项目储备和产量成长性的全球铜矿龙头之一。公司2024年矿产铜产量107万吨，2025年约109万吨，2026年规划120万吨，2028年目标150万-160万吨。公司在巨龙铜矿、塞尔维亚铜金项目、卡莫阿-卡库拉权益产量、秘鲁La Arena等项目上的持续推进，使其具备穿越铜周期的扩张能力。

### 3. 锂与钼：可选成长曲线开始形成

相较市场过去主要聚焦“金+铜”双轮驱动，紫金矿业中期更值得重视的是锂与钼业务的成长曲线。公司规划当量碳酸锂产量由2025年2.5万吨提升至2026年12万吨、2028年270万-320万吨区间中的27万-32万吨；矿产钼产量则由2025年1.1万吨提升至2026年1.5万吨、2028年2.5万-3.5万吨。

这意味着公司正在从“全球化金铜龙头”进一步走向“多金属成长平台”。尽管锂价波动较大，但一旦行业价格在未来企稳，公司在产量弹性上的估值重估空间将进一步打开。

## 五、投资逻辑与推荐理由

### 1. 从周期股向成长型资源平台重估

传统资源股估值往往受制于“价格驱动、产量稳定、周期波动大”的框架，而紫金矿业当前更接近“商品价格上行+产量持续释放+全球并购整合”三重驱动。公司增长不再完全依赖商品价格，而是具有明确的产量成长路径。

### 2. 全球化资源配置能力构筑壁垒

公司在海外项目并购、建设和运营上的成功案例显著多于多数A股资源企业，且形成了较强的工程、技术、管理和成本控制闭环。对中国上市矿企而言，这种能力稀缺且难以复制，理应享受高于传统资源股的估值溢价。

### 3. 2025-2026年盈利弹性突出

基于公司已披露的2025年初步数据与2026年产量规划，我们测算：  
- 2025E归母净利润约515亿元；  
- 2026E归母净利润约610亿元。  

上述测算为**基于公司已披露的2025年预增区间、2026年产量目标以及金铜价格中枢延续高位的推演**，并非公司盈利指引。按总股本265.78亿股估算，2025E/2026E EPS分别约1.94元/2.30元。

给予18.3倍2026E PE，主要考虑：  
1. 公司兼具黄金防御属性与铜成长属性；  
2. 2025-2026年产量释放确定性较高；  
3. 全球龙头地位提升及资本运作能力增强，有望推动估值中枢高于一般有色周期股。  

以2026年3月12日A股收盘价36.75元计算，目标价对应约14.3%的上行空间，因此给予“买入”评级。

## 七、风险因素

1. **金价、铜价大幅回落风险**：若全球实际利率回升、风险偏好修复或经济走弱超预期，可能导致金铜价格回调，压缩盈利弹性。  
2. **海外项目经营与地缘政治风险**：公司海外资产占比较高，资源国税制、环保、安全、劳工和政治环境变化均可能影响项目回报。  
3. **项目投产进度不及预期**：巨龙、塞尔维亚、锂资源项目及其他扩建项目若建设、爬坡延迟，将影响未来两年产量兑现。  
4. **并购整合风险**：新并购项目或控股平台若整合进度低于预期，可能拖累成本和现金流表现。  
5. **汇率及融资风险**：公司跨境资产、外币融资与海外收入占比较高，汇率波动及境外融资环境变化可能影响利润表和现金流。  
6. **ESG与安全生产风险**：矿业企业天然面临尾矿库、环保、社区关系和安全生产约束，一旦出现重大事件，估值与经营均可能承压。

## 八、结论

紫金矿业已经从中国矿业龙头迈向全球资源平台型公司。站在2026年时点，公司最突出的特征不是“资源股弹性”，而是“全球铜金龙头+多金属扩张平台”的成长属性。我们认为，若金铜价格维持高位、公司扩产按计划推进，市场对其估值框架仍有上修空间。

维持**买入**评级，A股**12个月目标价42.00元**。
